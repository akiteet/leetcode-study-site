https://akiteet.github.io/leetcode-study-site/

## 本地打开

下载或克隆仓库后，直接用浏览器打开：

```text
index.html
```

不需要后端服务，也不需要安装依赖。


## 更新方式

本地修改 HTML、CSS 或 JavaScript 后：

1. 刷新本地浏览器查看效果。
2. 在 GitHub Desktop 中提交修改。
3. 推送到 GitHub。
4. GitHub Pages 会自动更新线上网站。
