    73. Set Matrix Zeroes

    如果某个位置是 0，就把它所在的整行和整列都变成 0。

    Medium矩阵数组

    题目白话解释

    给你一个二维矩阵，只要矩阵里某个格子是 0，这个格子所在的整行、整列最后都要变成 0。关键是不能在扫描时立刻改，否则新改出来的 0 会继续影响其他行列。

    核心思路一句话

    先记录原始矩阵中哪些行、哪些列需要清零，再统一修改。

    为什么会想到这个方法

    题目要求根据“原来的 0”决定结果。如果边扫边改，会把后来改出来的 0 当成原始 0，导致清零范围扩大，所以要先做标记。

    Python 前置知识

    • 二维列表可以用 matrix[i][j] 访问。
    • set() 适合记录需要处理的行号和列号。
    • 先收集信息、后统一修改，是矩阵题常见思路。

    解题步骤

    1. 遍历矩阵，遇到 0 就把行号放进 rows，列号放进 cols。
    2. 再次遍历矩阵。
    3. 如果当前位置的行号在 rows，或者列号在 cols，就改成 0。
    4. 原地修改矩阵，不需要返回新矩阵。

    完整 Python 代码

    leetcode.py
    class Solution:
    def setZeroes(self, matrix):
        rows = set()
        cols = set()

        for i in range(len(matrix)):
            for j in range(len(matrix[0])):
                if matrix[i][j] == 0:
                    rows.add(i)
                    cols.add(j)

        for i in range(len(matrix)):
            for j in range(len(matrix[0])):
                if i in rows or j in cols:
                    matrix[i][j] = 0

    逐行解释

    • rows 保存需要清零的行。
    • cols 保存需要清零的列。
    • 第一次双层循环只做记录，不修改矩阵。
    • 第二次双层循环根据标记统一清零。

    示例执行过程

    [[1,1,1],[1,0,1],[1,1,1]] 中间位置是 0，所以第 1 行和第 1 列都变成 0。

    时间复杂度

    O(mn)，两次遍历矩阵。

    空间复杂度

    O(m+n)，记录行号和列号。

    易错点

    • 不要在第一次扫描时直接清零。
    • 列数要用 len(matrix[0])，不要误写成行数。
    • 题目要求原地修改，不要只返回新矩阵。

    ACM 模式输入输出改写

    ACM 模式可以先读入 m n，再读入 m 行矩阵，处理后逐行输出。

    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
m, n = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(m)]

rows = set()
cols = set()
for i in range(m):
    for j in range(n):
        if matrix[i][j] == 0:
            rows.add(i)
            cols.add(j)

for i in range(m):
    for j in range(n):
        if i in rows or j in cols:
            matrix[i][j] = 0

for row in matrix:
    print(*row)
