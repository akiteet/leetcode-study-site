    DFS / Matrix

    130. Surrounded Regions

    把被 X 完全包围的 O 改成 X。

    MediumDFSBFS矩阵

    题目白话解释

    如果一片 O 没有连到边界，就说明它被 X 包围，需要改成 X。和边界相连的 O 不能改。

    核心思路一句话

    从边界 O 出发标记安全区域，剩下未标记的 O 全部翻转。

    为什么会想到这个方法

    直接找“被包围”的区域不容易，但“不被包围”的 O 一定能连到边界。反过来从边界找安全 O，问题就简单了。

    Python 前置知识

    • 二维矩阵可原地修改。
    • DFS 可从边界向内扩散。
    • 临时标记如 "S" 可表示安全。

    解题步骤

    1. 扫描四条边界。
    2. 遇到边界 O 就 DFS 标记为安全。
    3. 遍历整张棋盘。
    4. 未标记的 O 改成 X。
    5. 安全标记 S 改回 O。

    完整 Python 代码

    leetcode.py
    class Solution:
    def solve(self, board: list[list[str]]) -> None:
        if not board:
            return

        rows, cols = len(board), len(board[0])

        def dfs(r, c):
            if r < 0 or r >= rows or c < 0 or c >= cols or board[r][c] != "O":
                return
            board[r][c] = "S"
            dfs(r + 1, c)
            dfs(r - 1, c)
            dfs(r, c + 1)
            dfs(r, c - 1)

        for r in range(rows):
            dfs(r, 0)
            dfs(r, cols - 1)
        for c in range(cols):
            dfs(0, c)
            dfs(rows - 1, c)

        for r in range(rows):
            for c in range(cols):
                if board[r][c] == "O":
                    board[r][c] = "X"
                elif board[r][c] == "S":
                    board[r][c] = "O"

    逐行解释

    • dfs：标记从边界能到达的安全 O。
    • board[r][c] = "S"：临时安全标记。
    • 先处理左右边界，再处理上下边界。
    • 剩余 O 是被包围区域。
    • S 最后恢复为 O。

    示例执行过程

    如果中间一片 O 没有连接到边界，它不会被 DFS 标记为 S，最后会被改成 X；边界连通的 O 会保留。

    时间复杂度

    O(m * n)，每个格子最多访问常数次。

    空间复杂度

    O(m * n)，最坏情况下递归栈可能覆盖整张棋盘。

    易错点

    不要从内部 O 开始判断是否被包围。边界反向标记更简单，也更不容易漏情况。

    ACM 模式输入输出改写

    输入 m n，后面 m 行每行为 X/O 字符串，输出修改后的棋盘。

    acm.py
    import sys

def solve(board):
    rows, cols = len(board), len(board[0])
    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols or board[r][c] != "O":
            return
        board[r][c] = "S"
        dfs(r + 1, c)
        dfs(r - 1, c)
        dfs(r, c + 1)
        dfs(r, c - 1)
    for r in range(rows):
        dfs(r, 0)
        dfs(r, cols - 1)
    for c in range(cols):
        dfs(0, c)
        dfs(rows - 1, c)
    for r in range(rows):
        for c in range(cols):
            if board[r][c] == "O":
                board[r][c] = "X"
            elif board[r][c] == "S":
                board[r][c] = "O"

lines = sys.stdin.read().strip().splitlines()
m, n = map(int, lines[0].split())
board = [list(lines[i + 1].strip()) for i in range(m)]
solve(board)
for row in board:
    print("".join(row))
