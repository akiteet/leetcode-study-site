    Dynamic Programming

    279. Perfect Squares

    求和为 n 的完全平方数的最少数量。

    Medium动态规划入门完全背包

    题目白话解释

    比如 12 可以拆成 4 + 4 + 4，所以最少需要 3 个完全平方数。

    核心思路一句话

    设 dp[i] 为组成 i 需要的最少平方数，枚举最后使用的平方数。

    为什么会想到这个方法

    如果最后用了 square，那么问题变成 i - square 的最优解加 1，符合从小问题推大问题。

    Python 前置知识

    • 完全平方数有 1, 4, 9, 16...。
    • float('inf') 可以表示初始无穷大。
    • 动态规划需要先确定状态含义。

    解题步骤

    1. 列出不超过 n 的所有平方数。
    2. 初始化 dp[0] = 0，其他为无穷大。
    3. 从 1 遍历到 n。
    4. 枚举每个不超过当前值的平方数。
    5. 更新 dp[i] = min(dp[i], dp[i-square] + 1)。

    完整 Python 代码

    leetcode.py
    class Solution:
    def numSquares(self, n: int) -> int:
        squares = []
        x = 1
        while x * x <= n:
            squares.append(x * x)
            x += 1

        dp = [float("inf")] * (n + 1)
        dp[0] = 0
        for i in range(1, n + 1):
            for square in squares:
                if square > i:
                    break
                dp[i] = min(dp[i], dp[i - square] + 1)
        return dp[n]

    逐行解释

    • squares 保存可选的平方数。
    • dp[0] = 0 是递推起点。
    • dp[i-square] + 1 表示最后选了一个 square。
    • 取最小值就是最少数量。

    示例执行过程

    n = 13 时，13 = 4 + 9，所以答案是 2。

    时间复杂度

    O(n√n)。

    空间复杂度

    O(n)。

    易错点

    • 不是求有多少种组合，而是求最少个数。
    • 平方数可以重复使用。
    • dp[0] 必须初始化为 0。
