    Dynamic Programming

    139. Word Break

    判断字符串能否由字典中的单词拼接而成。

    Medium动态规划入门字符串哈希表

    题目白话解释

    给你一个字符串和一个单词表，问能不能把字符串切成若干段，每段都在单词表中。

    核心思路一句话

    设 dp[i] 表示前 i 个字符能否被成功拆分。

    为什么会想到这个方法

    如果前 j 个字符可以拆分，并且 s[j:i] 在字典里，那么前 i 个字符也可以拆分。

    Python 前置知识

    • 字符串切片 s[j:i] 表示左闭右开。
    • 用 set 查询单词是否存在更快。
    • 布尔 DP 常用 True / False 表示可达。

    解题步骤

    1. 把单词列表转成集合。
    2. 初始化 dp[0] = True，表示空串可拆分。
    3. 枚举结束位置 i。
    4. 枚举切分位置 j。
    5. 如果 dp[j] 为真且 s[j:i] 在集合中，设置 dp[i] = True。

    完整 Python 代码

    leetcode.py
    class Solution:
    def wordBreak(self, s: str, wordDict: list[str]) -> bool:
        words = set(wordDict)
        dp = [False] * (len(s) + 1)
        dp[0] = True

        for i in range(1, len(s) + 1):
            for j in range(i):
                if dp[j] and s[j:i] in words:
                    dp[i] = True
                    break

        return dp[len(s)]

    逐行解释

    • dp[i] 只关心前缀 s[:i]。
    • dp[0] = True 是所有拆分的起点。
    • s[j:i] 是最后一个单词候选。
    • 找到一个合法切法就可以 break。

    示例执行过程

    s=&#x27;leetcode&#x27;，字典 [&#x27;leet&#x27;,&#x27;code&#x27;]，可以拆成 leet + code。

    时间复杂度

    O(n^2 * L)，切片和查询会带来字符串长度成本。

    空间复杂度

    O(n + m)，DP 数组和单词集合。

    易错点

    • dp[0] 必须是 True。
    • 切片右边界 i 不包含自身。
    • 单词可以重复使用。

    ACM 模式输入输出改写

    输入字符串，再输入一行词典，用空格分隔，输出是否可拆分。
