    Sliding Window / Hash Table

    3. Longest Substring Without Repeating Characters

    求不含重复字符的最长连续子串长度。

    Medium 滑动窗口 哈希表 字符串

    核心思路

    用左指针维护窗口起点，用字典记录字符最近出现的位置。遇到重复字符时，把左指针移动到重复字符上一次位置之后。

    题目理解

    • 子串必须连续，子序列不算。
    • 窗口内不能有重复字符。
    • 需要返回长度，不需要返回具体字符串。

    解题过程

    1. 设置左边界 left = 0。
    2. 用字典 last_seen 保存字符最近下标。
    3. 遍历右边界 right。
    4. 如果字符在窗口内重复，移动左边界。
    5. 更新答案和字符最近位置。

    Python 代码块

    leetcode.py
    class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last_seen = {}
        left = 0
        best = 0

        for right, ch in enumerate(s):
            if ch in last_seen and last_seen[ch] >= left:
                left = last_seen[ch] + 1

            last_seen[ch] = right
            best = max(best, right - left + 1)

        return best

    逐行解释

    • last_seen：保存每个字符最后一次出现的位置。
    • left：当前无重复窗口的左边界。
    • last_seen[ch] >= left：重复字符必须在当前窗口内才需要移动。
    • left = last_seen[ch] + 1：跳过重复字符上一次出现的位置。
    • right - left + 1：当前窗口长度。

    复杂度分析

    时间复杂度 O(n)，每个字符最多让左右指针各经过一次。空间复杂度 O(k)，k 是字符种类数。

    易错点

    左边界只能向右移动，不能被窗口外的旧重复字符拉回去，所以要判断 last_seen[ch] >= left。

    ACM 模式改写

    输入一行字符串，输出最长无重复子串长度。

    acm.py
    import sys

def longest_substring(s):
    last_seen = {}
    left = 0
    best = 0
    for right, ch in enumerate(s):
        if ch in last_seen and last_seen[ch] >= left:
            left = last_seen[ch] + 1
        last_seen[ch] = right
        best = max(best, right - left + 1)
    return best

s = sys.stdin.readline().rstrip("\n")
print(longest_substring(s))
