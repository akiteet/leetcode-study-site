    763. Partition Labels

    把字符串划分成尽可能多的片段，使每个字母最多出现在一个片段中。

    Medium贪心字符串

    题目白话解释

    如果某个字母在片段里出现了，那么这个字母最后一次出现的位置也必须被包含进这个片段。

    核心思路一句话

    先记录每个字母最后出现的位置，遍历时维护当前片段必须覆盖到的最远边界。

    为什么会想到这个方法

    片段能结束的前提是，片段中所有字符的最后出现位置都已经被包含。这个最远位置就是贪心边界。

    Python 前置知识

    • 字典可以记录字符最后出现的下标。
    • enumerate(s) 可以同时拿到下标和字符。
    • 边界闭合时记录片段长度。

    解题步骤

    1. 遍历字符串，记录每个字符最后出现位置。
    2. 再次遍历，更新当前片段右边界。
    3. 如果当前位置等于右边界，说明片段可以结束。
    4. 记录长度，并把下一段起点设为 i + 1。

    完整 Python 代码

    leetcode.py
    class Solution:
    def partitionLabels(self, s: str) -> list[int]:
        last = {ch: i for i, ch in enumerate(s)}
        ans = []
        start = 0
        end = 0

        for i, ch in enumerate(s):
            end = max(end, last[ch])
            if i == end:
                ans.append(end - start + 1)
                start = i + 1

        return ans

    逐行解释

    • last 保存每个字符的最后位置。
    • end 是当前片段最远必须覆盖到的位置。
    • 当 i == end 时，片段内字符不会再出现在后面。
    • 长度是 end - start + 1。

    示例执行过程

    ababcbacadefegdehijhklij 的划分长度是 [9,7,8]。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)，字符集固定时最多 26 个字母。

    易错点

    • 不能只看当前字符是否重复。
    • 片段边界要取所有字符最后位置的最大值。
    • 结果输出的是每段长度，不是字符串片段。

    ACM 模式输入输出改写

    输入字符串，输出每段长度。

    # input() 读取字符串；split() / map() 常用于数组输入；print() 输出答案

    end = max(end, last[ch])
