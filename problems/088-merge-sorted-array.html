    Two Pointers / Array

    88. Merge Sorted Array

    把两个非递减数组合并到 nums1 中。

    Easy 双指针 数组

    核心思路

    从后往前合并。因为 nums1 末尾有空位，先放较大的数不会覆盖还没处理的数据。

    题目理解

    • nums1 的长度是 m + n，前 m 个是有效数字。
    • nums2 的长度是 n。
    • 要求原地修改 nums1，不需要返回值。

    解题过程

    1. p1 指向 nums1 有效部分末尾。
    2. p2 指向 nums2 末尾。
    3. write 指向 nums1 总末尾。
    4. 每次把较大的数字写到 write 位置。
    5. 如果 nums2 还有剩余，继续写入。

    Python 代码块

    leetcode.py
    class Solution:
    def merge(self, nums1: list[int], m: int, nums2: list[int], n: int) -> None:
        p1, p2 = m - 1, n - 1
        write = m + n - 1

        while p2 >= 0:
            if p1 >= 0 and nums1[p1] > nums2[p2]:
                nums1[write] = nums1[p1]
                p1 -= 1
            else:
                nums1[write] = nums2[p2]
                p2 -= 1
            write -= 1

    逐行解释

    • p1, p2：分别指向两个数组当前最大元素。
    • write：当前要写入的位置。
    • while p2 >= 0：只要 nums2 还没放完就继续。
    • nums1[p1] > nums2[p2]：较大值放到后面。
    • 如果 nums1 剩余，不用额外处理，因为它已经在正确位置。

    复杂度分析

    时间复杂度 O(m + n)。空间复杂度 O(1)。

    易错点

    不要从前往后写，否则可能覆盖 nums1 中还没有比较的有效元素。

    ACM 模式改写

    输入 m n，第二行 m 个数，第三行 n 个数，输出合并后的数组。

    acm.py
    import sys

def merge(nums1, nums2):
    m, n = len(nums1), len(nums2)
    nums1.extend([0] * n)
    p1, p2 = m - 1, n - 1
    write = m + n - 1
    while p2 >= 0:
        if p1 >= 0 and nums1[p1] > nums2[p2]:
            nums1[write] = nums1[p1]
            p1 -= 1
        else:
            nums1[write] = nums2[p2]
            p2 -= 1
        write -= 1
    return nums1

data = sys.stdin.read().strip().split()
m, n = map(int, data[:2])
nums1 = list(map(int, data[2:2 + m]))
nums2 = list(map(int, data[2 + m:2 + m + n]))
print(*merge(nums1, nums2))
