    49. Group Anagrams

    把由相同字母组成的字符串分到同一组。

    Medium 哈希表 字符串

    核心思路

    字母异位词排序后会得到相同字符串。用排序结果作为字典 key，把原字符串加入对应分组。

    题目理解

    • "eat"、"tea"、"ate" 属于同一组。
    • 输出组的顺序通常不严格要求。
    • 关键是为每个字符串生成稳定且可比较的分组标识。

    解题过程

    1. 创建 defaultdict(list) 保存分组。
    2. 遍历每个字符串，把它排序后转成 key。
    3. 把原字符串追加到对应 key 的列表中。
    4. 返回字典中所有分组。

    Python 代码块

    leetcode.py
    from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs: list[str]) -> list[list[str]]:
        groups = defaultdict(list)

        for word in strs:
            key = "".join(sorted(word))
            groups[key].append(word)

        return list(groups.values())

    逐行解释

    • defaultdict(list)：当 key 不存在时自动创建空列表。
    • sorted(word)：把单词按字符排序，异位词会得到相同结果。
    • "".join(...)：把字符列表转成字符串，方便作为字典 key。
    • groups[key].append(word)：保留原字符串，并放进对应分组。
    • list(groups.values())：只需要返回每个分组，不需要返回 key。

    复杂度分析

    设字符串数量为 n，最长字符串长度为 k。排序法时间复杂度 O(n * k log k)，空间复杂度 O(n * k)。

    易错点

    不要用原字符串本身做 key。异位词字符顺序不同，原字符串不同，但排序后的签名相同。

    ACM 模式改写

    假设输入第一项为字符串数量 n，后面跟着 n 个单词。每行输出一个分组，组内用空格分隔。

    acm.py
    import sys
from collections import defaultdict

def group_anagrams(words):
    groups = defaultdict(list)
    for word in words:
        key = "".join(sorted(word))
        groups[key].append(word)
    return list(groups.values())

data = sys.stdin.read().strip().split()
n = int(data[0])
words = data[1:1 + n]

for group in group_anagrams(words):
    print(" ".join(group))
