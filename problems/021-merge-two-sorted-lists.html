    Linked List

    21. Merge Two Sorted Lists

    把两个升序链表合并成一个新的升序链表。

    Easy链表递归

    题目白话解释

    每次从两个链表当前头节点里选择较小的一个接到结果后面。

    核心思路一句话

    用虚拟头节点简化头部处理，尾指针不断接较小节点。

    为什么会想到这个方法

    如果没有虚拟头节点，第一次接节点时需要单独判断结果链表是否为空。虚拟头让所有节点都按同一种方式追加。

    Python 前置知识

    • dummy 是链表题常用辅助节点。
    • tail 指向结果链表最后一个节点。
    • 有一个链表为空后，剩余部分可以整体接上。

    解题步骤

    1. 创建 dummy 和 tail。
    2. 当两个链表都非空时，比较当前节点值。
    3. 把较小节点接到 tail.next。
    4. 移动被选中的链表指针和 tail。
    5. 最后接上剩余链表，返回 dummy.next。

    完整 Python 代码

    leetcode.py
    class Solution:
    def mergeTwoLists(self, list1, list2):
        dummy = ListNode(0)
        tail = dummy

        while list1 and list2:
            if list1.val <= list2.val:
                tail.next = list1
                list1 = list1.next
            else:
                tail.next = list2
                list2 = list2.next
            tail = tail.next

        tail.next = list1 if list1 else list2
        return dummy.next

    逐行解释

    • dummy.next 才是真正结果头。
    • tail 每次接上一个较小节点。
    • 比较使用 <= 可以保持相对稳定。
    • 循环结束后，非空链表剩下部分已经有序，直接接上。

    示例执行过程

    1-&gt;2-&gt;4 和 1-&gt;3-&gt;4 合并成 1-&gt;1-&gt;2-&gt;3-&gt;4-&gt;4。

    时间复杂度

    O(m+n)。

    空间复杂度

    O(1)，只调整指针。

    易错点

    • 返回 dummy.next，不要返回 dummy。
    • 移动 tail 后不要忘记移动原链表指针。
    • 最后要接剩余链表。

    ACM 模式输入输出改写

    数组输入时可以用双指针合并两个有序序列。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
a = list(map(int, input().split()))
b = list(map(int, input().split()))

i = j = 0
ans = []
while i < len(a) and j < len(b):
    if a[i] <= b[j]:
        ans.append(a[i])
        i += 1
    else:
        ans.append(b[j])
        j += 1
ans.extend(a[i:])
ans.extend(b[j:])
print(*ans)
