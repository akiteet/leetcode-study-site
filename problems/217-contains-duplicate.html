    217. Contains Duplicate

    判断数组中是否存在重复元素。

    Easy 哈希表 数组

    核心思路

    集合天然去重。遍历数组时，如果当前数字已经在集合中，说明存在重复；否则加入集合继续检查。

    题目理解

    • 只需要判断是否存在重复，不需要找出重复数字。
    • 一旦发现重复就可以立即返回 True。
    • 空数组或只有一个元素时一定没有重复。

    解题过程

    1. 创建空集合 seen。
    2. 逐个遍历数组元素。
    3. 如果元素已经在集合中，返回 True。
    4. 否则加入集合。
    5. 遍历结束仍未发现重复，返回 False。

    Python 代码块

    leetcode.py
    class Solution:
    def containsDuplicate(self, nums: list[int]) -> bool:
        seen = set()

        for num in nums:
            if num in seen:
                return True
            seen.add(num)

        return False

    逐行解释

    • seen = set()：用集合记录已经出现过的数字。
    • for num in nums：逐个检查每个数字。
    • if num in seen：集合查找平均是 O(1)。
    • return True：发现重复后可以提前结束。
    • seen.add(num)：当前数字第一次出现，记录下来。

    复杂度分析

    时间复杂度 O(n)，每个元素最多处理一次。空间复杂度 O(n)，最坏情况下所有元素都不同，需要全部存入集合。

    易错点

    如果写成 len(nums) != len(set(nums)) 也可以，但遍历写法能在发现重复时提前返回，更适合讲清楚哈希表思路。

    ACM 模式改写

    假设输入第一项为 n，后面跟着 n 个整数。存在重复输出 true，否则输出 false。

    acm.py
    import sys

def contains_duplicate(nums):
    seen = set()
    for num in nums:
        if num in seen:
            return True
        seen.add(num)
    return False

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))

print("true" if contains_duplicate(nums) else "false")
