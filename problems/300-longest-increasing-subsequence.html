    Dynamic Programming

    300. Longest Increasing Subsequence

    求数组中最长严格递增子序列的长度。

    Medium动态规划入门数组

    题目白话解释

    子序列不要求连续，但要保持原来的相对顺序，并且每个数都比前一个大。

    核心思路一句话

    入门 DP：dp[i] 表示以 nums[i] 结尾的最长递增子序列长度。

    为什么会想到这个方法

    如果 nums[j] &lt; nums[i]，那么以 j 结尾的递增子序列可以接上 nums[i]，形成更长序列。

    Python 前置知识

    • 子序列不等于子数组，子数组必须连续。
    • 严格递增用 <，不是 <=。
    • 每个元素单独也能形成长度 1 的序列。

    解题步骤

    1. 初始化所有 dp[i] = 1。
    2. 枚举当前位置 i。
    3. 枚举它前面的所有位置 j。
    4. 如果 nums[j] < nums[i]，尝试用 dp[j] + 1 更新。
    5. 答案是 max(dp)。

    完整 Python 代码

    leetcode.py
    class Solution:
    def lengthOfLIS(self, nums: list[int]) -> int:
        dp = [1] * len(nums)

        for i in range(len(nums)):
            for j in range(i):
                if nums[j] < nums[i]:
                    dp[i] = max(dp[i], dp[j] + 1)

        return max(dp)

    逐行解释

    • dp[i] 固定以 nums[i] 作为结尾。
    • 只有前面更小的数才能接到当前数前面。
    • dp[j] + 1 表示把当前数接到 j 的序列后面。
    • 最长序列可以结尾在任意位置，所以取最大值。

    示例执行过程

    [10,9,2,5,3,7,101,18] 的 LIS 长度是 4，例如 [2,3,7,101]。

    时间复杂度

    O(n^2)，入门版双层循环。

    空间复杂度

    O(n)。

    易错点

    • 子序列可以不连续。
    • 严格递增不能把相等数字接上。
    • 答案不是 dp[-1]，而是 max(dp)。

    ACM 模式输入输出改写

    输入数组，输出最长递增子序列长度。

    acm.py
    # input() + split() + map() 读取数组，print() 输出答案
nums = list(map(int, input().split()))

dp = [1] * len(nums)
for i in range(len(nums)):
    for j in range(i):
        if nums[j] < nums[i]:
            dp[i] = max(dp[i], dp[j] + 1)

print(max(dp))
