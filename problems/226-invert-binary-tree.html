    Binary Tree / DFS

    226. Invert Binary Tree

    把二叉树每个节点的左右子树交换。

    Easy 二叉树 DFS

    核心思路

    递归访问每个节点，交换它的左右孩子，然后继续翻转左右子树。

    题目理解

    • 翻转是对整棵树的每个节点都做左右交换。
    • 空树返回空。
    • 可以前序、后序或层序实现，关键是每个节点交换一次。

    解题过程

    1. 如果节点为空，返回 None。
    2. 交换当前节点的左右孩子。
    3. 递归翻转新的左子树。
    4. 递归翻转新的右子树。
    5. 返回根节点。

    Python 代码块

    leetcode.py
    class Solution:
    def invertTree(self, root):
        if root is None:
            return None

        root.left, root.right = root.right, root.left
        self.invertTree(root.left)
        self.invertTree(root.right)

        return root

    逐行解释

    • root is None：递归出口。
    • root.left, root.right = ...：交换当前节点的左右孩子。
    • self.invertTree(root.left)：继续处理左侧子树。
    • self.invertTree(root.right)：继续处理右侧子树。
    • return root：返回翻转后的根节点。

    复杂度分析

    时间复杂度 O(n)，每个节点交换一次。空间复杂度 O(h)，来自递归栈。

    易错点

    交换后再递归也可以，因为左右子树都要翻转；不要只交换根节点。

    ACM 模式改写

    假设输入层序数组，空节点用 null 表示。输出翻转后的数组位置表示。

    acm.py
    import sys
from collections import deque

class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def build_tree(values):
    if not values or values[0] == "null":
        return None
    root = TreeNode(values[0])
    queue = deque([root])
    i = 1
    while queue and i < len(values):
        node = queue.popleft()
        if i < len(values) and values[i] != "null":
            node.left = TreeNode(values[i])
            queue.append(node.left)
        i += 1
        if i < len(values) and values[i] != "null":
            node.right = TreeNode(values[i])
            queue.append(node.right)
        i += 1
    return root

def invert(root):
    if root is None:
        return None
    root.left, root.right = root.right, root.left
    invert(root.left)
    invert(root.right)
    return root

def serialize(root):
    if root is None:
        return []
    ans = []
    queue = deque([root])
    while queue:
        node = queue.popleft()
        if node is None:
            ans.append("null")
            continue
        ans.append(str(node.val))
        queue.append(node.left)
        queue.append(node.right)
    while ans and ans[-1] == "null":
        ans.pop()
    return ans

values = sys.stdin.read().strip().split()
root = invert(build_tree(values))
print(*serialize(root))
