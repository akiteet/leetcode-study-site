    Binary Tree / DFS

    112. Path Sum

    判断是否存在一条从根到叶子的路径，使路径节点和等于目标值。

    Easy二叉树DFS

    题目白话解释

    从根节点一路走到某个叶子节点，把路上的值加起来。如果能刚好等于目标值，就返回 True。

    核心思路一句话

    递归向下扣减目标值，到叶子节点时检查剩余值是否等于叶子值。

    为什么会想到这个方法

    路径必须从根到叶子。每走到一个节点，就相当于目标和减少这个节点值，问题变成“子树里有没有路径凑出剩余值”。

    Python 前置知识

    • 叶子节点满足 root.left is None 且 root.right is None。
    • 递归可以把目标值作为参数传下去。
    • or 表示左右子树有一个成功即可。

    解题步骤

    1. 空节点返回 False。
    2. 判断当前节点是否为叶子节点。
    3. 如果是叶子，检查 targetSum == root.val。
    4. 否则递归左右子树，目标值减去当前节点值。

    完整 Python 代码

    leetcode.py
    class Solution:
    def hasPathSum(self, root, targetSum: int) -> bool:
        if root is None:
            return False

        if root.left is None and root.right is None:
            return targetSum == root.val

        remain = targetSum - root.val
        return self.hasPathSum(root.left, remain) or self.hasPathSum(root.right, remain)

    逐行解释

    • 空树没有路径。
    • 叶子节点是路径结束位置。
    • 到叶子时才判断路径和。
    • remain 表示子树还需要凑出的值。
    • 左右子树任意一边成功即可。

    示例执行过程

    目标值 22，路径 5 -> 4 -> 11 -> 2 的和正好是 22，因此返回 True。

    时间复杂度

    O(n)，最坏情况下访问所有节点。

    空间复杂度

    O(h)，递归栈高度为树高。

    易错点

    必须是根到叶子的完整路径，不能在中间节点凑够目标值就提前返回 True。

    ACM 模式输入输出改写

    第一行输入层序数组，第二行输入目标值。

    acm.py
    import sys
from collections import deque

class TreeNode:
    def __init__(self, val):
        self.val = int(val)
        self.left = None
        self.right = None

def build_tree(values):
    if not values or values[0] == "null":
        return None
    root = TreeNode(values[0])
    queue = deque([root])
    i = 1
    while queue and i < len(values):
        node = queue.popleft()
        if i < len(values) and values[i] != "null":
            node.left = TreeNode(values[i])
            queue.append(node.left)
        i += 1
        if i < len(values) and values[i] != "null":
            node.right = TreeNode(values[i])
            queue.append(node.right)
        i += 1
    return root

def has_path_sum(root, target):
    if root is None:
        return False
    if root.left is None and root.right is None:
        return target == root.val
    return has_path_sum(root.left, target - root.val) or has_path_sum(root.right, target - root.val)

lines = sys.stdin.read().strip().splitlines()
root = build_tree(lines[0].split())
target = int(lines[1])
print("true" if has_path_sum(root, target) else "false")
