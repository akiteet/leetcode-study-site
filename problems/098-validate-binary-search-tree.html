    Binary Tree

    98. Validate Binary Search Tree

    判断一棵树是否满足二叉搜索树规则。

    Medium二叉树二叉搜索树DFS

    题目白话解释

    每个节点不仅要大于左孩子、小于右孩子，还要满足整个左子树都小于它，整个右子树都大于它。

    核心思路一句话

    递归时传入当前节点允许的取值范围。

    为什么会想到这个方法

    只比较父子节点不够，因为右子树里的某个深层节点也可能小于根。上下界能把祖先限制一起传下来。

    Python 前置知识

    • BST 的限制来自所有祖先节点。
    • 可以用 float('-inf') 和 float('inf') 表示初始边界。
    • 递归返回布尔值。

    解题步骤

    1. 定义 dfs(node, low, high)。
    2. 空节点返回 True。
    3. 如果节点值不在 (low, high) 内，返回 False。
    4. 左子树上界变为当前值。
    5. 右子树下界变为当前值。

    完整 Python 代码

    leetcode.py
    class Solution:
    def isValidBST(self, root):
        def dfs(node, low, high):
            if not node:
                return True
            if not (low < node.val < high):
                return False
            return dfs(node.left, low, node.val) and dfs(node.right, node.val, high)

        return dfs(root, float("-inf"), float("inf"))

    逐行解释

    • low 是当前节点必须大于的下界。
    • high 是当前节点必须小于的上界。
    • 左子树不能大于等于当前节点。
    • 右子树不能小于等于当前节点。

    示例执行过程

    [5,1,4,null,null,3,6] 不是 BST，因为 3 在 5 的右子树里却小于 5。

    时间复杂度

    O(n)。

    空间复杂度

    O(h)。

    易错点

    • 不能只比较直接左右孩子。
    • BST 通常不允许重复值，所以是严格小于和严格大于。
    • 边界要随递归更新。

    ACM 模式输入输出改写

    层序数组输入时，可用下标递归模拟树并验证上下界。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
vals = input().split()
# 常用整数行：nums = list(map(int, input().split()))

def dfs(i, low, high):
    if i >= len(vals) or vals[i] == "null":
        return True
    val = int(vals[i])
    if not (low < val < high):
        return False
    return dfs(2 * i + 1, low, val) and dfs(2 * i + 2, val, high)

print(dfs(0, float("-inf"), float("inf")))
