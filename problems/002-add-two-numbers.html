    Linked List

    2. Add Two Numbers

    两个链表按逆序存储数字，求和后也按逆序链表返回。

    Medium链表模拟

    题目白话解释

    2 -&gt; 4 -&gt; 3 表示 342，5 -&gt; 6 -&gt; 4 表示 465，结果是 807，即 7 -&gt; 0 -&gt; 8。

    核心思路一句话

    像小学竖式加法一样，从低位开始逐位相加，并维护进位。

    为什么会想到这个方法

    链表本身就是从个位到高位排列，正好适合从低位开始计算，不需要先反转。

    Python 前置知识

    • divmod(total, 10) 可以同时得到进位和当前位。
    • 两个链表长度可能不同。
    • 最后如果还有进位，要新建一个节点。

    解题步骤

    1. 创建虚拟头和 carry = 0。
    2. 当任一链表未结束或还有进位时继续循环。
    3. 取两个当前位，没有节点就当 0。
    4. 计算总和、当前位和新进位。
    5. 追加新节点并移动指针。

    完整 Python 代码

    leetcode.py
    class Solution:
    def addTwoNumbers(self, l1, l2):
        dummy = ListNode(0)
        tail = dummy
        carry = 0

        while l1 or l2 or carry:
            x = l1.val if l1 else 0
            y = l2.val if l2 else 0
            carry, digit = divmod(x + y + carry, 10)
            tail.next = ListNode(digit)
            tail = tail.next

            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None

        return dummy.next

    逐行解释

    • x 和 y 是当前位数字。
    • carry 保存进位。
    • digit 是当前节点要放的数字。
    • while l1 or l2 or carry 能处理最后一位进位。

    示例执行过程

    [2,4,3] + [5,6,4]：个位 2+5=7，十位 4+6=10 写 0 进 1，百位 3+4+1=8。

    时间复杂度

    O(max(m,n))。

    空间复杂度

    O(max(m,n))，结果链表长度。

    易错点

    • 不要忘记最后的 carry。
    • 链表是逆序存储，不要按正常数字方向处理。
    • 两个链表长度不同，空位要当作 0。

    ACM 模式输入输出改写

    ACM 中可用两行数组表示逆序数字，输出逆序结果。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
a = list(map(int, input().split()))
b = list(map(int, input().split()))

i = j = carry = 0
ans = []
while i < len(a) or j < len(b) or carry:
    x = a[i] if i < len(a) else 0
    y = b[j] if j < len(b) else 0
    carry, digit = divmod(x + y + carry, 10)
    ans.append(digit)
    i += 1
    j += 1

print(*ans)
