    20. Valid Parentheses

    判断括号字符串是否按正确顺序闭合。

    Easy 栈 字符串

    核心思路

    遇到左括号就入栈，遇到右括号就检查栈顶是否是对应的左括号。最后栈为空才说明全部匹配。

    题目理解

    • 括号必须类型匹配，例如 ()、[]、{}。
    • 括号还必须顺序正确，例如 (] 不合法。
    • 右括号出现时，如果没有可匹配的左括号，也不合法。

    解题过程

    1. 用字典保存右括号到左括号的对应关系。
    2. 遍历字符串，左括号入栈。
    3. 遇到右括号，栈为空则失败。
    4. 弹出栈顶，检查是否匹配当前右括号。
    5. 遍历结束后，栈为空则成功。

    Python 代码块

    leetcode.py
    class Solution:
    def isValid(self, s: str) -> bool:
        pairs = {")": "(", "]": "[", "}": "{"}
        stack = []

        for ch in s:
            if ch in pairs.values():
                stack.append(ch)
            else:
                if not stack or stack.pop() != pairs[ch]:
                    return False

        return not stack

    逐行解释

    • pairs：用右括号查它应该匹配的左括号。
    • stack.append(ch)：左括号暂时等待后续右括号匹配。
    • not stack：右括号前没有左括号，直接失败。
    • stack.pop()：拿最近的左括号匹配当前右括号。
    • return not stack：剩余左括号没有闭合，也是不合法。

    复杂度分析

    时间复杂度 O(n)，空间复杂度 O(n)，最坏情况下所有字符都是左括号。

    易错点

    不能只统计左右括号数量。"([)]" 数量正确，但闭合顺序错误。

    ACM 模式改写

    输入一行括号字符串，输出 true 或 false。

    acm.py
    import sys

def is_valid(s):
    pairs = {")": "(", "]": "[", "}": "{"}
    stack = []
    for ch in s:
        if ch in pairs.values():
            stack.append(ch)
        elif ch in pairs:
            if not stack or stack.pop() != pairs[ch]:
                return False
    return not stack

s = sys.stdin.readline().strip()
print("true" if is_valid(s) else "false")
