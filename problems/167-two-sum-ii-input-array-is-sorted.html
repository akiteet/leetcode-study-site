    Two Pointers

    167. Two Sum II - Input Array Is Sorted

    在有序数组中找两个数，使它们的和等于目标值。

    Medium双指针数组

    题目白话解释

    数组已经从小到大排好序，返回两个数字的位置。注意 LeetCode 原题返回的是从 1 开始的下标。

    核心思路一句话

    用左右指针，根据当前和偏大或偏小来收缩范围。

    为什么会想到这个方法

    有序数组提供了方向性：和太小就需要更大的数，移动左指针；和太大就需要更小的数，移动右指针。

    Python 前置知识

    • 数组下标从 0 开始，题目答案可能要加 1。
    • while left < right 是双指针常见写法。
    • 可以用列表 [left + 1, right + 1] 返回答案。

    解题步骤

    1. 左指针指向开头，右指针指向结尾。
    2. 计算当前两数之和。
    3. 等于目标值就返回下标。
    4. 小于目标值，左指针右移。
    5. 大于目标值，右指针左移。

    完整 Python 代码

    leetcode.py
    class Solution:
    def twoSum(self, numbers: list[int], target: int) -> list[int]:
        left, right = 0, len(numbers) - 1

        while left < right:
            total = numbers[left] + numbers[right]
            if total == target:
                return [left + 1, right + 1]
            if total < target:
                left += 1
            else:
                right -= 1

        return []

    逐行解释

    • left, right：从最小值和最大值开始。
    • total：当前两数和。
    • left + 1：题目要求返回 1-based 下标。
    • total < target：需要更大数字，所以左指针右移。
    • right -= 1：和太大，需要更小数字。

    示例执行过程

    numbers = [2,7,11,15], target = 9。左右指针指向 2 和 15，和太大，右移到 7 后和为 9，返回 [1,2]。

    时间复杂度

    O(n)，两个指针最多各走一遍数组。

    空间复杂度

    O(1)。

    易错点

    这题返回 1-based 下标，不是 Python 的 0-based 下标。

    ACM 模式输入输出改写

    输入第一项 n，接着 n 个有序整数，最后一个数是 target。

    acm.py
    import sys

def two_sum_sorted(numbers, target):
    left, right = 0, len(numbers) - 1
    while left < right:
        total = numbers[left] + numbers[right]
        if total == target:
            return left + 1, right + 1
        if total < target:
            left += 1
        else:
            right -= 1
    return -1, -1

data = sys.stdin.read().strip().split()
n = int(data[0])
numbers = list(map(int, data[1:1 + n]))
target = int(data[1 + n])
print(*two_sum_sorted(numbers, target))
