    1. Two Sum

    给定数组和目标值，返回两个数的下标，使它们相加等于目标值。

    Easy 哈希表 数组

    核心思路

    一边遍历数组，一边用哈希表记录已经见过的数字和下标。当前数字是 x，只要检查 target - x 是否已经出现即可。

    题目理解

    • 每个输入只对应一个答案。
    • 不能重复使用同一个元素。
    • 返回的是下标，不是数值本身。

    解题过程

    1. 创建字典 seen，保存数字到下标的映射。
    2. 遍历数组，计算当前数字需要的补数。
    3. 如果补数在字典里，说明找到答案。
    4. 否则把当前数字和下标存入字典，继续遍历。

    Python 代码块

    leetcode.py
    class Solution:
    def twoSum(self, nums: list[int], target: int) -> list[int]:
        seen = {}

        for i, num in enumerate(nums):
            need = target - num
            if need in seen:
                return [seen[need], i]
            seen[num] = i

        return []

    逐行解释

    • seen = {}：保存已经遍历过的数字和它的下标。
    • enumerate(nums)：同时拿到下标 i 和数字 num。
    • need = target - num：计算当前数字还需要配对的值。
    • if need in seen：如果补数已经出现，就返回两个下标。
    • seen[num] = i：当前数字暂时找不到配对，存起来给后面的数字用。

    复杂度分析

    时间复杂度 O(n)，每个元素最多查一次字典。空间复杂度 O(n)，最坏情况下需要保存所有元素。

    易错点

    先查补数，再存当前数字。否则当 target = 2 * num 时，可能错误地使用同一个元素两次。

    ACM 模式改写

    假设输入三行：第一行 n，第二行 n 个整数，第三行 target。输出两个下标。

    acm.py
    import sys

def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        need = target - num
        if need in seen:
            return seen[need], i
        seen[num] = i
    return -1, -1

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
target = int(data[1 + n])

ans = two_sum(nums, target)
print(ans[0], ans[1])
