    Sort / Two Pointers

    15. 3Sum

    找出所有不重复的三元组，使三个数相加等于 0。

    Medium双指针排序数组

    题目白话解释

    在数组中找三个人组队，三个人的数字加起来为 0。相同三元组只能出现一次。

    核心思路一句话

    先排序，固定一个数，再用双指针找另外两个数。

    为什么会想到这个方法

    三数之和可以拆成“固定一个数后的两数之和”。排序后，左右指针可以根据当前和偏大或偏小来移动，同时也方便跳过重复值。

    Python 前置知识

    • nums.sort() 原地排序。
    • 列表可以用 [a, b, c] 保存三元组。
    • continue 可以跳过重复固定值。

    解题步骤

    1. 排序数组。
    2. 枚举第一个数 nums[i]。
    3. 跳过重复的第一个数。
    4. 用 left 和 right 找剩余两数。
    5. 找到答案后移动指针并跳过重复值。

    完整 Python 代码

    leetcode.py
    class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        nums.sort()
        ans = []

        for i in range(len(nums) - 2):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            if nums[i] > 0:
                break

            left, right = i + 1, len(nums) - 1
            while left < right:
                total = nums[i] + nums[left] + nums[right]
                if total == 0:
                    ans.append([nums[i], nums[left], nums[right]])
                    left += 1
                    right -= 1
                    while left < right and nums[left] == nums[left - 1]:
                        left += 1
                    while left < right and nums[right] == nums[right + 1]:
                        right -= 1
                elif total < 0:
                    left += 1
                else:
                    right -= 1

        return ans

    逐行解释

    • nums.sort()：让指针移动有方向。
    • nums[i] == nums[i - 1]：避免重复固定同一个值。
    • nums[i] > 0：排序后第一个数都大于 0，后面不可能凑成 0。
    • total < 0：和太小，左指针右移变大。
    • 找到答案后跳过重复左右值。

    示例执行过程

    输入 [-1,0,1,2,-1,-4]，排序为 [-4,-1,-1,0,1,2]。固定第一个 -1 时，双指针能找到 [-1,-1,2] 和 [-1,0,1]。

    时间复杂度

    O(n^2)，外层枚举一个数，内层双指针线性扫描。

    空间复杂度

    除答案外为 O(1)；排序可能因实现使用 O(log n) 递归栈。

    易错点

    去重有三处：固定值去重，找到答案后左指针去重，找到答案后右指针去重。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着数组。每行输出一个三元组。

    acm.py
    import sys

def three_sum(nums):
    nums.sort()
    ans = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        left, right = i + 1, len(nums) - 1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total == 0:
                ans.append([nums[i], nums[left], nums[right]])
                left += 1
                right -= 1
                while left < right and nums[left] == nums[left - 1]:
                    left += 1
                while left < right and nums[right] == nums[right + 1]:
                    right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1
    return ans

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
for item in three_sum(nums):
    print(*item)
