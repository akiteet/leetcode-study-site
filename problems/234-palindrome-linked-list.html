    Linked List

    234. Palindrome Linked List

    判断链表从前往后和从后往前读是否一样。

    Easy链表快慢指针

    题目白话解释

    比如 1 -&gt; 2 -&gt; 2 -&gt; 1 是回文，1 -&gt; 2 不是回文。

    核心思路一句话

    快慢指针找到中点，反转后半段，再和前半段逐个比较。

    为什么会想到这个方法

    链表不能从尾部往前读，所以要把后半段反转，让它能从中间向后和前半段同步比较。

    Python 前置知识

    • 快指针一次走两步，慢指针一次走一步。
    • 快指针到尾部时，慢指针在中点附近。
    • 可以复用反转链表模板。

    解题步骤

    1. 用快慢指针找到链表中点。
    2. 从慢指针开始反转后半段。
    3. 用两个指针分别从头部和后半段头部开始比较。
    4. 有一个值不同就返回 False。
    5. 比较结束返回 True。

    完整 Python 代码

    leetcode.py
    class Solution:
    def isPalindrome(self, head):
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        prev = None
        cur = slow
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt

        left, right = head, prev
        while right:
            if left.val != right.val:
                return False
            left = left.next
            right = right.next
        return True

    逐行解释

    • 第一段 while 用来找中点。
    • 第二段 while 是反转链表模板。
    • right 是反转后的后半段。
    • 只要后半段比较完，就说明是回文。

    示例执行过程

    1 -&gt; 2 -&gt; 2 -&gt; 1 中点后反转得到 1 -&gt; 2，和前半段 1 -&gt; 2 相同。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 奇数长度时，中间节点不影响结果。
    • 比较循环以 right 为准。
    • 如果面试要求恢复链表，需要再反转一次后半段。

    ACM 模式输入输出改写

    数组形式输入链表时，可以直接用双指针比较首尾。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
nums = list(map(int, input().split()))

left, right = 0, len(nums) - 1
ok = True
while left < right:
    if nums[left] != nums[right]:
        ok = False
        break
    left += 1
    right -= 1

print(ok)
