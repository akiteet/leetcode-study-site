    Binary Tree

    114. Flatten Binary Tree to Linked List

    把二叉树按前序遍历顺序原地展开成右指针链表。

    Medium二叉树DFS链表

    题目白话解释

    展开后每个节点的左指针都为空，右指针指向前序遍历中的下一个节点。

    核心思路一句话

    用反向前序（右、左、根）遍历，维护上一个处理好的节点 prev。

    为什么会想到这个方法

    前序是根左右，如果从后往前处理，就能让当前节点的右指针直接指向已经处理好的后继节点。

    Python 前置知识

    • 前序遍历顺序是根、左、右。
    • 反向处理顺序是右、左、根。
    • nonlocal prev 保存展开后的后继节点。

    解题步骤

    1. 初始化 prev = None。
    2. 递归函数先处理右子树。
    3. 再处理左子树。
    4. 把当前节点右指针指向 prev。
    5. 左指针置空，并更新 prev 为当前节点。

    完整 Python 代码

    leetcode.py
    class Solution:
    def flatten(self, root):
        prev = None

        def dfs(node):
            nonlocal prev
            if not node:
                return
            dfs(node.right)
            dfs(node.left)
            node.right = prev
            node.left = None
            prev = node

        dfs(root)

    逐行解释

    • 先处理右再处理左，是为了反向拼接前序链表。
    • node.right = prev 把当前节点接到已展开链表前面。
    • node.left = None 满足题目要求。
    • prev 始终指向当前节点后面的链表头。

    示例执行过程

    [1,2,5,3,4,null,6] 展开后右链是 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 5 -&gt; 6。

    时间复杂度

    O(n)。

    空间复杂度

    O(h)。

    易错点

    • 不是中序或后序展开，而是前序。
    • 必须把左指针置空。
    • 如果正向处理，容易丢失右子树。

    ACM 模式输入输出改写

    ACM 可输出展开后的前序序列作为右链结果。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
vals = input().split()
# 常用整数行：nums = list(map(int, input().split()))

def preorder(i):
    if i >= len(vals) or vals[i] == "null":
        return []
    return [int(vals[i])] + preorder(2 * i + 1) + preorder(2 * i + 2)

print(*preorder(0))
