    Linked List

    24. Swap Nodes in Pairs

    每两个相邻节点交换一次，不能只交换节点值。

    Medium链表递归

    题目白话解释

    1 -&gt; 2 -&gt; 3 -&gt; 4 变成 2 -&gt; 1 -&gt; 4 -&gt; 3。

    核心思路一句话

    用虚拟头节点，每次拿出一组两个节点，重新连接三条边。

    为什么会想到这个方法

    链表交换节点本质是改 next 指向。虚拟头可以让第一组节点和后续组用同样逻辑处理。

    Python 前置知识

    • 三指针 prev、first、second 可以描述一组交换。
    • 交换前要保存 second.next。
    • 链表题画图会明显降低出错率。

    解题步骤

    1. 创建 dummy，prev 指向交换组前一个节点。
    2. 当 prev.next 和 prev.next.next 都存在时处理一组。
    3. 记录 first 和 second。
    4. 把 first.next 指向下一组开头。
    5. 把 second.next 指向 first，再把 prev.next 指向 second。
    6. 移动 prev = first。

    完整 Python 代码

    leetcode.py
    class Solution:
    def swapPairs(self, head):
        dummy = ListNode(0, head)
        prev = dummy

        while prev.next and prev.next.next:
            first = prev.next
            second = first.next

            first.next = second.next
            second.next = first
            prev.next = second

            prev = first

        return dummy.next

    逐行解释

    • first 是一组中的第一个节点。
    • second 是一组中的第二个节点。
    • 三次指针修改完成交换。
    • 交换后 first 变成这一组尾部，所以下一轮 prev = first。

    示例执行过程

    1,2 交换后 2 指向 1；接着处理 3,4。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 题目要求交换节点，不是交换值。
    • 奇数长度时最后一个节点保持不变。
    • 三条边修改顺序不要写乱。

    ACM 模式输入输出改写

    数组输入时可以每两个元素交换一次来模拟输出。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
nums = list(map(int, input().split()))

for i in range(0, len(nums) - 1, 2):
    nums[i], nums[i + 1] = nums[i + 1], nums[i]

print(*nums)
