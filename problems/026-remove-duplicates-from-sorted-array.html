    Two Pointers / Array

    26. Remove Duplicates from Sorted Array

    在有序数组中原地保留每个元素第一次出现的位置。

    Easy 双指针 数组

    核心思路

    数组已经有序，重复元素一定相邻。用慢指针记录唯一元素写入位置，用快指针扫描新元素。

    题目理解

    • 只需要返回去重后的长度 k。
    • 前 k 个位置必须是去重结果。
    • 数组后面的内容不重要。

    解题过程

    1. 空数组直接返回 0。
    2. 设置 write = 1，因为第一个元素一定保留。
    3. 从第二个元素开始扫描。
    4. 如果当前元素和前一个元素不同，写入 write 位置。
    5. 返回 write，它就是唯一元素数量。

    Python 代码块

    leetcode.py
    class Solution:
    def removeDuplicates(self, nums: list[int]) -> int:
        if not nums:
            return 0

        write = 1
        for read in range(1, len(nums)):
            if nums[read] != nums[read - 1]:
                nums[write] = nums[read]
                write += 1

        return write

    逐行解释

    • if not nums：处理空数组边界。
    • write = 1：第一个元素默认保留。
    • nums[read] != nums[read - 1]：有序数组中新值只需要和前一个位置比较。
    • nums[write] = nums[read]：把新的唯一元素写到前面。
    • return write：写指针走过的位置数量就是答案。

    复杂度分析

    时间复杂度 O(n)，空间复杂度 O(1)。

    易错点

    这题依赖数组已经有序。如果数组无序，不能只比较相邻元素。

    ACM 模式改写

    输入第一项为 n，后面跟着 n 个有序整数。输出长度和去重后的前缀。

    acm.py
    import sys

def remove_duplicates(nums):
    if not nums:
        return 0
    write = 1
    for read in range(1, len(nums)):
        if nums[read] != nums[read - 1]:
            nums[write] = nums[read]
            write += 1
    return write

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
k = remove_duplicates(nums)
print(k)
print(*nums[:k])
