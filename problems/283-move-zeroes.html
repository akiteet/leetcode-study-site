    Two Pointers / Array

    283. Move Zeroes

    把数组中的 0 移到末尾，同时保持非零元素相对顺序。

    Easy 双指针 数组

    核心思路

    用一个写指针记录下一个非零元素应该放的位置。遍历数组，把非零元素依次前移，最后把剩余位置补 0。

    题目理解

    • 需要原地修改数组。
    • 非零元素的相对顺序不能变。
    • 不需要返回新数组。

    解题过程

    1. 设置 write = 0。
    2. 遍历数组，遇到非零数字就写到 write 位置。
    3. 每写入一个非零数字，write 后移。
    4. 遍历结束后，从 write 到末尾全部写成 0。

    Python 代码块

    leetcode.py
    class Solution:
    def moveZeroes(self, nums: list[int]) -> None:
        write = 0

        for num in nums:
            if num != 0:
                nums[write] = num
                write += 1

        for i in range(write, len(nums)):
            nums[i] = 0

    逐行解释

    • write：下一个非零元素应该写入的位置。
    • for num in nums：按原顺序扫描，保证非零元素顺序不变。
    • nums[write] = num：把非零元素压到前面。
    • range(write, len(nums))：剩下的位置都应该是 0。

    复杂度分析

    时间复杂度 O(n)，空间复杂度 O(1)。

    易错点

    不能直接排序，因为排序会改变非零元素的相对顺序。

    ACM 模式改写

    输入第一项为 n，后面跟着 n 个整数，输出移动后的数组。

    acm.py
    import sys

def move_zeroes(nums):
    write = 0
    for num in nums:
        if num != 0:
            nums[write] = num
            write += 1
    for i in range(write, len(nums)):
        nums[i] = 0
    return nums

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
print(*move_zeroes(nums))
