    Dynamic Programming

    53. Maximum Subarray

    找出和最大的连续子数组。

    Medium动态规划入门数组

    题目白话解释

    在数组里选一段连续区间，让这段区间的数字和最大。数组里可能有负数。

    核心思路一句话

    当前位置的最大连续和，要么从当前数重新开始，要么接在前一个连续和后面。

    为什么会想到这个方法

    子数组必须连续，所以走到当前位置时，只需要关心“前一段连续和是否值得继续带上”。如果前面是负贡献，就从当前数重新开始。

    Python 前置知识

    • max(num, current + num) 表示重新开始或继续连接。
    • 数组可能全是负数，不能把答案初始化为 0。
    • 遍历可从第二个元素开始。

    解题步骤

    1. 把当前连续和和答案都初始化为第一个元素。
    2. 从第二个元素开始遍历。
    3. 计算以当前位置结尾的最大连续和。
    4. 更新全局最大值。
    5. 遍历结束返回答案。

    完整 Python 代码

    leetcode.py
    class Solution:
    def maxSubArray(self, nums: list[int]) -> int:
        current = nums[0]
        best = nums[0]

        for num in nums[1:]:
            current = max(num, current + num)
            best = max(best, current)

        return best

    逐行解释

    • current：以当前位置结尾的最大连续和。
    • best：全局最大连续和。
    • max(num, current + num)：决定重新开始还是接着前面。
    • best = max(best, current)：更新答案。
    • 初始化为 nums[0] 可以处理全负数组。

    示例执行过程

    [-2,1,-3,4,-1,2,1,-5,4] 中，最大连续段是 [4,-1,2,1]，和为 6。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    答案不能初始化为 0，否则全负数组会错误返回 0。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着 n 个整数，输出最大子数组和。

    acm.py
    import sys

def max_sub_array(nums):
    current = nums[0]
    best = nums[0]
    for num in nums[1:]:
        current = max(num, current + num)
        best = max(best, current)
    return best

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
print(max_sub_array(nums))
