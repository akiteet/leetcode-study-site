    54. Spiral Matrix

    按右、下、左、上的顺序一圈一圈读取矩阵。

    Medium矩阵模拟

    题目白话解释

    把矩阵想成一层层边框，先读最外圈，再读里面一圈，直到所有数字都读完。

    核心思路一句话

    维护上下左右四条边界，每读完一条边就把边界向内收缩。

    为什么会想到这个方法

    螺旋顺序本质是固定方向的模拟。只要明确当前还能访问的矩形边界，就不用额外记录每个格子是否访问过。

    Python 前置知识

    • 列表 append() 用来收集答案。
    • while top <= bottom and left <= right 表示还有合法矩形。
    • 每走完一个方向，都要移动对应边界。

    解题步骤

    1. 初始化 top、bottom、left、right。
    2. 从左到右读取上边界，然后 top += 1。
    3. 从上到下读取右边界，然后 right -= 1。
    4. 如果仍有行，从右到左读取下边界。
    5. 如果仍有列，从下到上读取左边界。

    完整 Python 代码

    leetcode.py
    class Solution:
    def spiralOrder(self, matrix):
        ans = []
        top, bottom = 0, len(matrix) - 1
        left, right = 0, len(matrix[0]) - 1

        while top <= bottom and left <= right:
            for j in range(left, right + 1):
                ans.append(matrix[top][j])
            top += 1

            for i in range(top, bottom + 1):
                ans.append(matrix[i][right])
            right -= 1

            if top <= bottom:
                for j in range(right, left - 1, -1):
                    ans.append(matrix[bottom][j])
                bottom -= 1

            if left <= right:
                for i in range(bottom, top - 1, -1):
                    ans.append(matrix[i][left])
                left += 1

        return ans

    逐行解释

    • top/bottom/left/right 表示还没有读取的矩形范围。
    • 读取上边后，上边界下移。
    • 读取右边后，右边界左移。
    • 读下边和左边前要判断边界，避免单行或单列重复读取。

    示例执行过程

    [[1,2,3],[4,5,6],[7,8,9]] 的输出是 [1,2,3,6,9,8,7,4,5]。

    时间复杂度

    O(mn)，每个格子只进答案一次。

    空间复杂度

    O(1) 额外空间，不算返回结果。

    易错点

    • 单行矩阵容易重复读下边。
    • 单列矩阵容易重复读左边。
    • 边界移动顺序要和读取方向对应。

    ACM 模式输入输出改写

    读入矩阵后输出螺旋顺序，通常用空格隔开。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
m, n = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(m)]

ans = []
top, bottom = 0, m - 1
left, right = 0, n - 1
while top <= bottom and left <= right:
    for j in range(left, right + 1):
        ans.append(matrix[top][j])
    top += 1
    for i in range(top, bottom + 1):
        ans.append(matrix[i][right])
    right -= 1
    if top <= bottom:
        for j in range(right, left - 1, -1):
            ans.append(matrix[bottom][j])
        bottom -= 1
    if left <= right:
        for i in range(bottom, top - 1, -1):
            ans.append(matrix[i][left])
        left += 1

print(*ans)
