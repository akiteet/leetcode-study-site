    Matrix

    240. Search a 2D Matrix II

    矩阵每行、每列都递增，判断目标值是否存在。

    Medium矩阵二分思想

    题目白话解释

    每一行从左到右递增，每一列从上到下递增。要利用这个有序性，不要暴力扫完整个矩阵。

    核心思路一句话

    从右上角出发，大了就左移，小了就下移。

    为什么会想到这个方法

    右上角同时是当前行最大、当前列最小。和目标比较后，可以一次排除一整列或一整行。

    Python 前置知识

    • 二维矩阵边界判断：0 <= row < m 且 0 <= col < n。
    • 有序矩阵常从角落开始，而不是从中间开始。
    • 每一步都要能排除一块不可能区域。

    解题步骤

    1. 从 row = 0, col = n - 1 开始。
    2. 当前值等于目标，返回 True。
    3. 当前值大于目标，说明这一列下面更大，左移。
    4. 当前值小于目标，说明这一行左边更小，下移。
    5. 越界后返回 False。

    完整 Python 代码

    leetcode.py
    class Solution:
    def searchMatrix(self, matrix, target):
        m, n = len(matrix), len(matrix[0])
        row, col = 0, n - 1

        while row < m and col >= 0:
            if matrix[row][col] == target:
                return True
            if matrix[row][col] > target:
                col -= 1
            else:
                row += 1

        return False

    逐行解释

    • row, col 指向右上角。
    • 当前值太大，当前列都不可能，col -= 1。
    • 当前值太小，当前行都不可能，row += 1。
    • 每次移动都会缩小搜索范围。

    示例执行过程

    目标 5，在右上角开始比较，逐步向左或向下移动，最终找到就返回 True。

    时间复杂度

    O(m+n)，最多走 m 次下移和 n 次左移。

    空间复杂度

    O(1)。

    易错点

    • 不要把本题和 74 题的整体有序矩阵混淆。
    • 从左上角开始无法直接排除。
    • 空矩阵时要先保证输入合法，LeetCode 本题通常给非空矩阵。

    ACM 模式输入输出改写

    读入矩阵和目标值，输出 True 或 False。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
m, n, target = map(int, input().split())
matrix = [list(map(int, input().split())) for _ in range(m)]

row, col = 0, n - 1
found = False
while row < m and col >= 0:
    if matrix[row][col] == target:
        found = True
        break
    if matrix[row][col] > target:
        col -= 1
    else:
        row += 1

print(found)
