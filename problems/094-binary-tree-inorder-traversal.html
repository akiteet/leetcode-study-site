    94. Binary Tree Inorder Traversal

    按左子树、根节点、右子树的顺序遍历二叉树。

    Easy二叉树DFS栈

    题目白话解释

    中序遍历就是先看左边，再看自己，最后看右边。对二叉搜索树来说，中序遍历结果是升序。

    核心思路一句话

    递归写法最直观：左、根、右。

    为什么会想到这个方法

    树本身是递归结构，一个节点的左子树和右子树又是树，所以用递归描述遍历最自然。

    Python 前置知识

    • 递归函数可以接收当前节点。
    • 空节点直接返回。
    • ans.append(root.val) 表示访问根节点。

    解题步骤

    1. 定义 dfs(node)。
    2. 如果节点为空，返回。
    3. 递归遍历左子树。
    4. 把当前节点值加入答案。
    5. 递归遍历右子树。

    完整 Python 代码

    leetcode.py
    class Solution:
    def inorderTraversal(self, root):
        ans = []

        def dfs(node):
            if not node:
                return
            dfs(node.left)
            ans.append(node.val)
            dfs(node.right)

        dfs(root)
        return ans

    逐行解释

    • ans 保存遍历结果。
    • dfs(node.left) 先处理左子树。
    • ans.append(node.val) 访问根。
    • dfs(node.right) 最后处理右子树。

    示例执行过程

    树 [1,null,2,3] 的中序遍历结果是 [1,3,2]。

    时间复杂度

    O(n)。

    空间复杂度

    O(h)，递归栈高度，最坏 O(n)。

    易错点

    • 顺序是左根右，不是根左右。
    • 空节点一定要作为递归出口。
    • 不要把 ans 写成类变量，避免多次调用污染。

    ACM 模式输入输出改写

    ACM 中可用层序数组建树，再输出中序遍历。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
# 常用整数行：nums = list(map(int, input().split()))
vals = input().split()

def inorder(i):
    if i >= len(vals) or vals[i] == "null":
        return []
    return inorder(2 * i + 1) + [int(vals[i])] + inorder(2 * i + 2)

print(*inorder(0))
