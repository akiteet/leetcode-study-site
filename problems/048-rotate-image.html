    Matrix

    48. Rotate Image

    把 n x n 矩阵顺时针旋转 90 度，要求原地修改。

    Medium矩阵原地修改

    题目白话解释

    题目不是让你返回新矩阵，而是直接把原矩阵变成旋转后的样子。

    核心思路一句话

    先沿主对角线转置，再把每一行反转。

    为什么会想到这个方法

    顺时针旋转 90 度可以拆成两个简单操作：转置让行列交换，行反转把列顺序调整到正确方向。

    Python 前置知识

    • 交换两个元素可写成 a, b = b, a。
    • row.reverse() 会原地反转列表。
    • 只遍历对角线上方即可完成转置，避免交换两次。

    解题步骤

    1. 设 n = len(matrix)。
    2. 遍历 i < j 的位置，交换 matrix[i][j] 和 matrix[j][i]。
    3. 对每一行执行 reverse()。
    4. 矩阵就完成了顺时针旋转。

    完整 Python 代码

    leetcode.py
    class Solution:
    def rotate(self, matrix):
        n = len(matrix)

        for i in range(n):
            for j in range(i + 1, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

        for row in matrix:
            row.reverse()

    逐行解释

    • 第一层循环选择行。
    • j 从 i + 1 开始，表示只处理主对角线上方。
    • 转置后，原来的列变成行。
    • 每行反转后，方向变成顺时针 90 度。

    示例执行过程

    [[1,2,3],[4,5,6],[7,8,9]] 旋转后是 [[7,4,1],[8,5,2],[9,6,3]]。

    时间复杂度

    O(n^2)。

    空间复杂度

    O(1)，原地交换。

    易错点

    • 不要新建矩阵返回。
    • 转置时不要遍历全部位置，否则会换回来。
    • 顺时针是转置后反转每一行，逆时针不是这个顺序。

    ACM 模式输入输出改写

    输入方阵后，原地旋转并逐行输出。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
n = int(input())
matrix = [list(map(int, input().split())) for _ in range(n)]

for i in range(n):
    for j in range(i + 1, n):
        matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
for row in matrix:
    row.reverse()

for row in matrix:
    print(*row)
