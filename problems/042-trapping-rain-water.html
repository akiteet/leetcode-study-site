    Two Pointers / Prefix Max

    42. Trapping Rain Water

    给定柱子高度，计算下雨后最多能接多少水。

    Hard双指针数组前后缀

    题目白话解释

    每个数字是一根柱子的高度。水能停在某个位置，取决于它左边最高柱子和右边最高柱子中较矮的那个，再减去当前位置柱子高度。

    核心思路一句话

    用左右指针维护两边最高柱子，每次处理较低边，因为较低边已经决定了当前位置能接多少水。

    为什么会想到这个方法

    某个位置能接水的高度是 min(左侧最高, 右侧最高) - 当前高度。如果左侧最高小于右侧最高，那么左侧当前位置的上限已经由左侧最高决定，不需要再知道更远的右边细节，可以直接结算并移动左指针。

    Python 前置知识

    • left、right 可从数组两端向中间移动。
    • left_max 和 right_max 记录两侧目前见过的最高柱子。
    • max(a, b) 用于更新最高值。

    解题步骤

    1. 初始化左右指针和两侧最高值。
    2. 当 left < right 时循环。
    3. 更新左右两侧当前最高柱子。
    4. 哪边最高值更小，就处理哪边。
    5. 当前位置能接的水等于该侧最高值减当前位置高度。
    6. 移动被处理的一侧指针。

    完整 Python 代码

    leetcode.py
    class Solution:
    def trap(self, height: list[int]) -> int:
        left, right = 0, len(height) - 1
        left_max = 0
        right_max = 0
        water = 0

        while left < right:
            left_max = max(left_max, height[left])
            right_max = max(right_max, height[right])

            if left_max < right_max:
                water += left_max - height[left]
                left += 1
            else:
                water += right_max - height[right]
                right -= 1

        return water

    逐行解释

    • left, right：分别从最左和最右开始。
    • left_max：左侧目前最高柱子。
    • right_max：右侧目前最高柱子。
    • left_max < right_max：左侧较矮，当前位置水位由左侧决定。
    • water += left_max - height[left]：累加左侧当前位置能接的水。
    • 右侧同理，用 right_max - height[right] 结算。

    示例执行过程

    输入 [0,1,0,2,1,0,1,3,2,1,2,1]。中间低洼位置会被两边较高柱子围住，例如高度 0 的位置左边最高为 2、右边最高为 3，所以这里能接 2 格水。最终总水量为 6。

    时间复杂度

    O(n)，左右指针总共只走完整个数组一次。

    空间复杂度

    O(1)，只使用左右指针和几个变量。

    易错点

    不要用当前左右柱子的高度直接判断接水量，要用两侧“目前最高柱子”。当前位置能接多少水取决于边界最高值，不是相邻柱子。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着 n 个柱子高度，输出能接的总水量。

    acm.py
    import sys

def trap(height):
    left, right = 0, len(height) - 1
    left_max = 0
    right_max = 0
    water = 0
    while left < right:
        left_max = max(left_max, height[left])
        right_max = max(right_max, height[right])
        if left_max < right_max:
            water += left_max - height[left]
            left += 1
        else:
            water += right_max - height[right]
            right -= 1
    return water

data = sys.stdin.read().strip().split()
n = int(data[0])
height = list(map(int, data[1:1 + n]))
print(trap(height))
