    Greedy / Array

    121. Best Time to Buy and Sell Stock

    只允许买入一次、卖出一次，求最大利润。

    Easy 贪心 数组

    核心思路

    遍历每天价格，维护历史最低买入价。当天卖出的最大利润就是 price - min_price。

    题目理解

    • 必须先买后卖，不能倒序交易。
    • 最多完成一笔交易。
    • 如果没有利润，返回 0。

    解题过程

    1. 初始化最低价格为正无穷。
    2. 初始化最大利润为 0。
    3. 遍历价格，先更新最低买入价。
    4. 再计算当前价格卖出的利润。
    5. 更新最大利润。

    Python 代码块

    leetcode.py
    class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        min_price = float("inf")
        best = 0

        for price in prices:
            min_price = min(min_price, price)
            best = max(best, price - min_price)

        return best

    逐行解释

    • min_price：当前天之前和当天能看到的最低买入价。
    • best：当前找到的最大利润。
    • min(min_price, price)：遇到更低价格就更新买入机会。
    • price - min_price：如果今天卖出能赚多少。
    • return best：价格一直下降时仍返回 0。

    复杂度分析

    时间复杂度 O(n)，空间复杂度 O(1)。

    易错点

    不能简单用最大值减最小值，因为最大值必须出现在最小值之后。

    ACM 模式改写

    输入第一项为 n，后面跟着 n 个价格，输出最大利润。

    acm.py
    import sys

def max_profit(prices):
    min_price = float("inf")
    best = 0
    for price in prices:
        min_price = min(min_price, price)
        best = max(best, price - min_price)
    return best

data = sys.stdin.read().strip().split()
n = int(data[0])
prices = list(map(int, data[1:1 + n]))
print(max_profit(prices))
