    394. Decode String

    把形如 3[a2[c]] 的压缩字符串还原成 accaccacc。

    Medium栈字符串

    题目白话解释

    数字表示后面括号里的字符串要重复多少次，括号可以嵌套，所以要先处理最里面的括号。

    核心思路一句话

    遇到左括号就把当前数字和当前字符串入栈，遇到右括号就弹出并拼接。

    为什么会想到这个方法

    括号嵌套符合后进先出的特点，最后遇到的左括号要最先被配对，所以自然想到栈。

    Python 前置知识

    • str.isdigit() 判断字符是否是数字。
    • 多位数字要用 num = num * 10 + int(ch) 累积。
    • 栈可以保存进入新括号前的状态。

    解题步骤

    1. 遍历字符串中的每个字符。
    2. 遇到数字就累积重复次数。
    3. 遇到 [，把当前字符串和数字入栈，然后清空当前状态。
    4. 遇到 ]，弹出上一层状态，把当前字符串重复后接回去。
    5. 遇到普通字母就追加到当前字符串。

    完整 Python 代码

    leetcode.py
    class Solution:
    def decodeString(self, s: str) -> str:
        stack = []
        current = ""
        num = 0

        for ch in s:
            if ch.isdigit():
                num = num * 10 + int(ch)
            elif ch == "[":
                stack.append((current, num))
                current = ""
                num = 0
            elif ch == "]":
                prev, times = stack.pop()
                current = prev + current * times
            else:
                current += ch

        return current

    逐行解释

    • current 表示当前层已经解码出的字符串。
    • num 表示当前括号前的重复次数。
    • 入栈保存进入括号前的 current 和 num。
    • 出栈时把当前层结果重复 times 次并接回上一层。

    示例执行过程

    3[a2[c]] 先把 2[c] 解成 cc，再得到 3[acc]，结果是 accaccacc。

    时间复杂度

    O(n + ans)，ans 是展开后字符串长度。

    空间复杂度

    O(n + ans)，栈和结果字符串占用空间。

    易错点

    • 不要只处理一位数字，12[a] 也合法。
    • 遇到 [ 后要把 num 清零。
    • 嵌套括号要靠栈保存上一层状态。
