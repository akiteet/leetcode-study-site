    Prefix Sum / Hash Table

    560. Subarray Sum Equals K

    统计和为 k 的连续子数组个数。

    Medium哈希表前缀和数组

    题目白话解释

    在数组里找所有连续区间，只要这个区间的和等于 k，就计数一次。

    核心思路一句话

    用哈希表记录前缀和出现次数，当前前缀和为 prefix 时，查 prefix - k 出现过几次。

    为什么会想到这个方法

    区间和可以转成两个前缀和的差：sum(i..j) = prefix[j] - prefix[i - 1]。要让区间和为 k，就需要以前出现过 prefix - k。

    Python 前置知识

    • dict 可以记录某个前缀和出现次数。
    • count.get(key, 0) 能安全读取默认值。
    • 前缀和从 0 开始，表示空前缀。

    解题步骤

    1. 初始化 prefix = 0，并记录前缀和 0 出现 1 次。
    2. 遍历数组，不断累加当前前缀和。
    3. 查找 prefix - k 出现次数并加入答案。
    4. 把当前前缀和次数加 1。
    5. 遍历结束返回答案。

    完整 Python 代码

    leetcode.py
    class Solution:
    def subarraySum(self, nums: list[int], k: int) -> int:
        count = {0: 1}
        prefix = 0
        ans = 0

        for num in nums:
            prefix += num
            ans += count.get(prefix - k, 0)
            count[prefix] = count.get(prefix, 0) + 1

        return ans

    逐行解释

    • {0: 1}：处理从下标 0 开始的区间。
    • prefix += num：得到当前位置前缀和。
    • prefix - k：如果以前有这个前缀和，中间区间和就是 k。
    • ans += ...：可能有多个相同前缀和，都要计入。
    • 最后记录当前前缀和，给后面使用。

    示例执行过程

    nums = [1,1,1], k = 2。前缀和依次为 1、2、3，能找到两个区间：前两个 1 和后两个 1。

    时间复杂度

    O(n)，数组只遍历一次。

    空间复杂度

    O(n)，最坏情况下前缀和都不同。

    易错点

    这题数组可能有负数，不能用普通滑动窗口；窗口和不再单调变化。

    ACM 模式输入输出改写

    输入第一行 n k，第二行 n 个整数，输出区间个数。

    acm.py
    import sys

def subarray_sum(nums, k):
    count = {0: 1}
    prefix = 0
    ans = 0
    for num in nums:
        prefix += num
        ans += count.get(prefix - k, 0)
        count[prefix] = count.get(prefix, 0) + 1
    return ans

data = sys.stdin.read().strip().split()
n, k = map(int, data[:2])
nums = list(map(int, data[2:2 + n]))
print(subarray_sum(nums, k))
