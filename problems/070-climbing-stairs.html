    Dynamic Programming

    70. Climbing Stairs

    每次爬 1 或 2 阶，爬到第 n 阶共有多少种方法。

    Easy动态规划入门

    题目白话解释

    到达第 n 阶的最后一步，要么从第 n-1 阶爬 1 阶上来，要么从第 n-2 阶爬 2 阶上来。

    核心思路一句话

    dp[n] = dp[n - 1] + dp[n - 2]。

    为什么会想到这个方法

    问题问“有多少种方法”，并且大问题可以由前面两个小问题组合出来，这是动态规划的典型信号。

    Python 前置知识

    • 变量滚动可以替代完整 dp 数组。
    • for 循环适合从小台阶推到大台阶。
    • 多变量赋值 a, b = b, a + b 可同时更新。

    解题步骤

    1. 第 1 阶有 1 种方法。
    2. 第 2 阶有 2 种方法。
    3. 从第 3 阶开始，答案等于前两阶之和。
    4. 用两个变量滚动保存前两项。

    完整 Python 代码

    leetcode.py
    class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n

        prev2, prev1 = 1, 2
        for _ in range(3, n + 1):
            current = prev1 + prev2
            prev2, prev1 = prev1, current

        return prev1

    逐行解释

    • n <= 2：基础情况。
    • prev2：表示到前两阶的方法数。
    • prev1：表示到前一阶的方法数。
    • current：当前阶方法数。
    • 循环结束后 prev1 就是第 n 阶答案。

    示例执行过程

    n = 4：第 1 阶 1 种，第 2 阶 2 种，第 3 阶 3 种，第 4 阶 5 种。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)，只保留两个历史状态。

    易错点

    第 2 阶是 2 种，不是 1 种：可以 1+1，也可以 2。

    ACM 模式输入输出改写

    输入一个整数 n，输出爬楼梯方法数。

    acm.py
    import sys

def climb_stairs(n):
    if n <= 2:
        return n
    prev2, prev1 = 1, 2
    for _ in range(3, n + 1):
        current = prev1 + prev2
        prev2, prev1 = prev1, current
    return prev1

data = sys.stdin.read().strip().split()
n = int(data[0])
print(climb_stairs(n))
