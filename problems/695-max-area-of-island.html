    695. Max Area of Island

    在 0/1 网格中找出面积最大的岛屿。

    MediumDFSBFS矩阵

    题目白话解释

    连续相邻的 1 组成一座岛，面积就是岛上 1 的数量。要找最大的那座岛有多少格。

    核心思路一句话

    遇到未访问陆地就 DFS，把整座岛走完并统计格子数。

    为什么会想到这个方法

    岛屿问题本质是连通块问题。DFS 很适合从一个点出发，把与它连通的所有点全部访问。

    Python 前置知识

    • 二维数组用 grid[r][c] 访问。
    • 递归函数可以返回当前连通块大小。
    • 访问过的陆地可以直接改成 0。

    解题步骤

    1. 遍历每个格子。
    2. 遇到 1 就从这里启动 DFS。
    3. DFS 中越界或遇到 0 返回 0。
    4. 把当前格改成 0，防止重复访问。
    5. 返回当前格面积 1 加上四个方向面积。

    完整 Python 代码

    leetcode.py
    class Solution:
    def maxAreaOfIsland(self, grid: list[list[int]]) -> int:
        rows, cols = len(grid), len(grid[0])

        def dfs(r, c):
            if r < 0 or r >= rows or c < 0 or c >= cols:
                return 0
            if grid[r][c] == 0:
                return 0

            grid[r][c] = 0
            return 1 + dfs(r + 1, c) + dfs(r - 1, c) + dfs(r, c + 1) + dfs(r, c - 1)

        best = 0
        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == 1:
                    best = max(best, dfs(r, c))

        return best

    逐行解释

    • rows, cols：矩阵边界。
    • 越界返回 0。
    • 水域返回 0。
    • grid[r][c] = 0：标记已访问。
    • 四个方向面积相加得到整座岛面积。

    示例执行过程

    遇到一块 1 后，DFS 会向上下左右扩散，直到这座岛所有 1 都被改成 0，并返回这座岛的面积。

    时间复杂度

    O(m * n)，每个格子最多访问一次。

    空间复杂度

    O(m * n)，最坏情况下递归栈覆盖整张网格。

    易错点

    一定要在 DFS 中及时标记已访问，否则相邻陆地会互相递归，造成重复访问。

    ACM 模式输入输出改写

    输入第一行 m n，后面 m 行为 0/1 矩阵。

    acm.py
    import sys

def max_area(grid):
    rows, cols = len(grid), len(grid[0])
    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols or grid[r][c] == 0:
            return 0
        grid[r][c] = 0
        return 1 + dfs(r + 1, c) + dfs(r - 1, c) + dfs(r, c + 1) + dfs(r, c - 1)
    best = 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == 1:
                best = max(best, dfs(r, c))
    return best

lines = sys.stdin.read().strip().splitlines()
m, n = map(int, lines[0].split())
grid = [list(map(int, lines[i + 1].split())) for i in range(m)]
print(max_area(grid))
