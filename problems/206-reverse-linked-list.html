    Linked List

    206. Reverse Linked List

    把链表方向从 1 -&gt; 2 -&gt; 3 改成 3 -&gt; 2 -&gt; 1。

    Easy链表双指针

    题目白话解释

    每个节点的 next 指针原来指向后一个节点，现在要改成指向前一个节点。

    核心思路一句话

    用 prev 保存已经反转好的前半段，用 cur 扫描当前节点。

    为什么会想到这个方法

    链表不能像数组一样随机访问。反转时一旦改掉 cur.next，就会丢失后续节点，所以必须先保存 nxt。

    Python 前置知识

    • 链表操作常用 prev、cur、nxt 三个变量。
    • 改指针前先保存下一步位置。
    • None 是反转后新尾节点的 next。

    解题步骤

    1. 初始化 prev = None，cur = head。
    2. 循环中先保存 nxt = cur.next。
    3. 让 cur.next = prev 完成当前节点反转。
    4. 移动 prev = cur，cur = nxt。
    5. 循环结束返回 prev。

    完整 Python 代码

    leetcode.py
    class Solution:
    def reverseList(self, head):
        prev = None
        cur = head

        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt

        return prev

    逐行解释

    • prev 表示反转完成部分的头。
    • cur 表示正在处理的节点。
    • nxt 防止断链后找不到后面的节点。
    • 最后 cur 为空，prev 就是新头节点。

    示例执行过程

    1 -&gt; 2 -&gt; 3 处理完 1 后变成 1 -&gt; None，再把 2 指向 1，最后 3 指向 2。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 忘记保存 nxt 会丢失后半段。
    • 返回的是 prev，不是原来的 head。
    • 空链表也要能返回 None。

    ACM 模式输入输出改写

    ACM 常把链表输入成一行数组，输出反转后的值序列。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
nums = list(map(int, input().split()))
nums.reverse()
print(*nums)
