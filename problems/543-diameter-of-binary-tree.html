    543. Diameter of Binary Tree

    求二叉树中任意两个节点之间最长路径的边数。

    Easy二叉树DFS

    题目白话解释

    最长路径不一定经过根节点，可能完全在某个子树里。

    核心思路一句话

    DFS 求每个节点左右子树高度，同时用 left + right 更新答案。

    为什么会想到这个方法

    一条经过某节点的最长路径，就是左子树最大深度加右子树最大深度。遍历每个节点时顺手计算即可。

    Python 前置知识

    • 树的高度可以递归计算。
    • nonlocal 可以在内层函数修改外层变量。
    • 直径按边数计算，左右高度相加正好是边数。

    解题步骤

    1. 初始化 ans = 0。
    2. 定义 depth(node) 返回节点高度。
    3. 递归求左右高度。
    4. 用 left + right 更新直径。
    5. 返回 max(left, right) + 1。

    完整 Python 代码

    leetcode.py
    class Solution:
    def diameterOfBinaryTree(self, root):
        ans = 0

        def depth(node):
            nonlocal ans
            if not node:
                return 0
            left = depth(node.left)
            right = depth(node.right)
            ans = max(ans, left + right)
            return max(left, right) + 1

        depth(root)
        return ans

    逐行解释

    • depth 返回节点向下的最大节点数。
    • left + right 是经过当前节点的最长边数。
    • ans 记录全局最大直径。
    • 返回高度给父节点继续使用。

    示例执行过程

    [1,2,3,4,5] 中最长路径可以是 4-2-1-3，长度为 3 条边。

    时间复杂度

    O(n)。

    空间复杂度

    O(h)。

    易错点

    • 直径不一定经过根节点。
    • 答案是边数，不是节点数。
    • 更新答案的位置在拿到左右高度之后。

    ACM 模式输入输出改写

    层序输入建树后，递归计算直径并输出。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
vals = input().split()
# 常用整数行：nums = list(map(int, input().split()))

def depth(i):
    global ans
    if i >= len(vals) or vals[i] == "null":
        return 0
    left = depth(2 * i + 1)
    right = depth(2 * i + 2)
    ans = max(ans, left + right)
    return max(left, right) + 1

ans = 0
depth(0)
print(ans)
