    84. Largest Rectangle in Histogram

    给定柱子高度，找能组成的最大矩形面积。

    Hard栈单调栈

    题目白话解释

    每根柱子都可以作为矩形的最低高度，关键是找它能向左、向右延伸多远。

    核心思路一句话

    用单调递增栈，在遇到更矮柱子时结算被弹出柱子的最大面积。

    为什么会想到这个方法

    当右侧出现更矮柱子时，栈顶柱子的右边界就确定了；它左边界是弹出后新的栈顶。

    Python 前置知识

    • 单调栈保存下标，而不是只保存高度。
    • 在末尾补一个高度 0，可以强制结算所有柱子。
    • 宽度计算是 i - stack[-1] - 1。

    解题步骤

    1. 在高度数组前后各加一个 0，方便处理边界。
    2. 维护一个高度递增的下标栈。
    3. 当前高度小于栈顶高度时，弹出栈顶并计算面积。
    4. 弹出后新的栈顶是左边第一个更矮柱子。
    5. 不断更新最大面积。

    完整 Python 代码

    leetcode.py
    class Solution:
    def largestRectangleArea(self, heights: list[int]) -> int:
        heights = [0] + heights + [0]
        stack = []
        best = 0

        for i, height in enumerate(heights):
            while stack and heights[stack[-1]] > height:
                mid = stack.pop()
                width = i - stack[-1] - 1
                best = max(best, heights[mid] * width)
            stack.append(i)

        return best

    逐行解释

    • 补 0 后，所有柱子最终都会被弹出结算。
    • mid 是当前作为最低高度的柱子。
    • 右边界是当前下标 i 的前一个位置。
    • 左边界是弹出后栈顶的后一个位置。

    示例执行过程

    [2,1,5,6,2,3] 中高度 5 和 6 可以形成宽度 2 的矩形，面积 10。

    时间复杂度

    O(n)，每个下标最多入栈和出栈一次。

    空间复杂度

    O(n)，单调栈。

    易错点

    • 栈里放下标，方便算宽度。
    • 宽度不是 i - mid，而是 i - stack[-1] - 1。
    • 末尾不补 0 容易漏掉递增尾巴。

    ACM 模式输入输出改写

    输入第一行 n，第二行 n 个高度，输出最大矩形面积。

    # input() + split() + map() 读取数组，print() 输出答案
n = int(input())
heights = list(map(int, input().split()))

heights = [0] + heights + [0]
stack = []
best = 0
for i, height in enumerate(heights):
    while stack and heights[stack[-1]] > height:
        mid = stack.pop()
        width = i - stack[-1] - 1
        best = max(best, heights[mid] * width)
    stack.append(i)

print(best)
