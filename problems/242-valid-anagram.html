    Hash Table / String

    242. Valid Anagram

    判断两个字符串是否由完全相同的字符组成。

    Easy 哈希表 字符串

    核心思路

    两个字符串如果是字母异位词，长度相同，并且每个字符出现次数也相同。可以用计数哈希表比较。

    题目理解

    • "anagram" 和 "nagaram" 是有效异位词。
    • "rat" 和 "car" 字符计数不同。
    • 先判断长度，可以提前排除不可能情况。

    解题过程

    1. 如果两个字符串长度不同，直接返回 False。
    2. 遍历第一个字符串，统计每个字符出现次数。
    3. 遍历第二个字符串，逐个减少对应计数。
    4. 如果某个字符不存在或次数不够，返回 False。
    5. 全部抵消完成后返回 True。

    Python 代码块

    leetcode.py
    class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        if len(s) != len(t):
            return False

        count = {}
        for ch in s:
            count[ch] = count.get(ch, 0) + 1

        for ch in t:
            if ch not in count or count[ch] == 0:
                return False
            count[ch] -= 1

        return True

    逐行解释

    • len(s) != len(t)：长度不同一定不是异位词。
    • count.get(ch, 0) + 1：统计字符出现次数。
    • ch not in count：第二个字符串出现了第一个字符串没有的字符。
    • count[ch] == 0：字符出现次数已经被用完。
    • count[ch] -= 1：用第二个字符串抵消第一个字符串的计数。

    复杂度分析

    时间复杂度 O(n)。空间复杂度取决于字符种类，通常为 O(1)，泛化字符集下为 O(k)。

    易错点

    只比较字符集合是不够的，必须比较次数。例如 "aab" 和 "abb" 的字符集合相同，但不是异位词。

    ACM 模式改写

    假设输入两行字符串，输出 true 或 false。

    acm.py
    import sys

def is_anagram(s, t):
    if len(s) != len(t):
        return False
    count = {}
    for ch in s:
        count[ch] = count.get(ch, 0) + 1
    for ch in t:
        if ch not in count or count[ch] == 0:
            return False
        count[ch] -= 1
    return True

lines = sys.stdin.read().splitlines()
s = lines[0].strip()
t = lines[1].strip()
print("true" if is_anagram(s, t) else "false")
