    105. Construct Binary Tree from Preorder and Inorder Traversal

    根据前序遍历和中序遍历还原二叉树。

    Medium二叉树递归哈希表

    题目白话解释

    前序可以告诉你当前根节点是谁，中序可以告诉你左子树和右子树分别有哪些节点。

    核心思路一句话

    前序第一个元素是根，在中序中找到根的位置，递归划分左右子树。

    为什么会想到这个方法

    中序中根左边全是左子树，根右边全是右子树；前序按根左右排列，能持续提供每棵子树的根。

    Python 前置知识

    • 前序：根、左、右。
    • 中序：左、根、右。
    • 用哈希表记录中序下标，避免每次线性查找。

    解题步骤

    1. 建立 value -> inorder index 的字典。
    2. 用递归函数接收前序区间和中序区间。
    3. 前序区间第一个值是根。
    4. 在中序中找到根位置，计算左子树大小。
    5. 递归构造左右子树并返回根。

    完整 Python 代码

    leetcode.py
    class Solution:
    def buildTree(self, preorder, inorder):
        index = {value: i for i, value in enumerate(inorder)}

        def build(pre_l, pre_r, in_l, in_r):
            if pre_l > pre_r:
                return None
            root_val = preorder[pre_l]
            root = TreeNode(root_val)
            mid = index[root_val]
            left_size = mid - in_l

            root.left = build(pre_l + 1, pre_l + left_size, in_l, mid - 1)
            root.right = build(pre_l + left_size + 1, pre_r, mid + 1, in_r)
            return root

        return build(0, len(preorder) - 1, 0, len(inorder) - 1)

    逐行解释

    • preorder[pre_l] 是当前子树根。
    • mid 是根在中序中的位置。
    • left_size 决定前序中左子树占多少个元素。
    • 左右区间切清楚后递归即可。

    示例执行过程

    前序 [3,9,20,15,7]，中序 [9,3,15,20,7]，根是 3，左子树只有 9，右子树是 20、15、7。

    时间复杂度

    O(n)。

    空间复杂度

    O(n)，哈希表和递归栈。

    易错点

    • 不要在中序数组里反复 index() 查找，会退化到 O(n^2)。
    • 区间左右边界最容易写错。
    • 题目通常假设节点值不重复。

    ACM 模式输入输出改写

    读入两行遍历序列，输出构造后树的前序用于校验，真实平台可按要求改输出。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
preorder = list(map(int, input().split()))
inorder = list(map(int, input().split()))

index = {value: i for i, value in enumerate(inorder)}

def rebuild(pre_l, pre_r, in_l, in_r):
    if pre_l > pre_r:
        return []
    root = preorder[pre_l]
    mid = index[root]
    left_size = mid - in_l
    left = rebuild(pre_l + 1, pre_l + left_size, in_l, mid - 1)
    right = rebuild(pre_l + left_size + 1, pre_r, mid + 1, in_r)
    return [root] + left + right

print(*rebuild(0, len(preorder) - 1, 0, len(inorder) - 1))
