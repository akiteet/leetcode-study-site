    Binary Tree / DFS

    100. Same Tree

    判断两棵二叉树是否结构相同且节点值相同。

    Easy二叉树DFS

    题目白话解释

    两棵树要完全一样：当前位置要么都为空，要么都有节点且值相同，左右子树也要继续一样。

    核心思路一句话

    同时递归比较两棵树的当前节点、左子树和右子树。

    为什么会想到这个方法

    树的问题常常可以拆成“当前节点是否满足 + 子树是否满足”。相同的树本质上就是每个对应位置都相同。

    Python 前置知识

    • None 表示空节点。
    • 递归函数需要先写出口。
    • and 可以组合多个判断条件。

    解题步骤

    1. 如果两个节点都为空，返回 True。
    2. 如果只有一个为空，返回 False。
    3. 如果两个节点值不同，返回 False。
    4. 递归比较左子树和右子树。

    完整 Python 代码

    leetcode.py
    class Solution:
    def isSameTree(self, p, q) -> bool:
        if p is None and q is None:
            return True
        if p is None or q is None:
            return False
        if p.val != q.val:
            return False

        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)

    逐行解释

    • 两个都空：当前结构一致。
    • 只有一个空：结构不同。
    • 值不同：节点内容不同。
    • 最后递归比较左右子树。

    示例执行过程

    两棵树都是 [1,2,3] 时，根节点 1 相同，左孩子 2 相同，右孩子 3 相同，所以返回 True。

    时间复杂度

    O(n)，n 是两棵树中较小范围内需要比较的节点数。

    空间复杂度

    O(h)，h 是递归栈高度。

    易错点

    不能只比较层序数组长度和值，还要确保每个空节点位置也一致。

    ACM 模式输入输出改写

    输入两行层序数组，空节点用 null 表示，输出 true 或 false。

    acm.py
    import sys
from collections import deque

class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def build_tree(values):
    if not values or values[0] == "null":
        return None
    root = TreeNode(values[0])
    queue = deque([root])
    i = 1
    while queue and i < len(values):
        node = queue.popleft()
        if i < len(values) and values[i] != "null":
            node.left = TreeNode(values[i])
            queue.append(node.left)
        i += 1
        if i < len(values) and values[i] != "null":
            node.right = TreeNode(values[i])
            queue.append(node.right)
        i += 1
    return root

def same_tree(p, q):
    if p is None and q is None:
        return True
    if p is None or q is None:
        return False
    return p.val == q.val and same_tree(p.left, q.left) and same_tree(p.right, q.right)

lines = sys.stdin.read().strip().splitlines()
p = build_tree(lines[0].split())
q = build_tree(lines[1].split())
print("true" if same_tree(p, q) else "false")
