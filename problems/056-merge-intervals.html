    Greedy / Sort

    56. Merge Intervals

    合并所有重叠区间。

    Medium贪心排序数组

    题目白话解释

    给你一堆区间，如果两个区间有重叠，就把它们合成一个更大的区间。

    核心思路一句话

    先按左端点排序，再从左到右维护当前合并后的最后一个区间。

    为什么会想到这个方法

    排序后，可能和当前区间重叠的只会是后面的相邻区间。这样只需要比较当前区间起点和答案最后区间的终点。

    Python 前置知识

    • list.sort(key=lambda x: x[0]) 按区间左端点排序。
    • 答案列表中最后一个区间可用 ans[-1] 访问。
    • max 用于扩展右端点。

    解题步骤

    1. 按区间起点排序。
    2. 遍历每个区间。
    3. 如果答案为空或不重叠，直接加入。
    4. 如果重叠，更新最后区间的右端点。
    5. 返回答案。

    完整 Python 代码

    leetcode.py
    class Solution:
    def merge(self, intervals: list[list[int]]) -> list[list[int]]:
        intervals.sort(key=lambda item: item[0])
        ans = []

        for start, end in intervals:
            if not ans or start > ans[-1][1]:
                ans.append([start, end])
            else:
                ans[-1][1] = max(ans[-1][1], end)

        return ans

    逐行解释

    • 先排序，让区间按起点从小到大排列。
    • not ans：第一个区间直接加入。
    • start > ans[-1][1]：当前区间在最后区间右侧，不重叠。
    • 否则说明重叠，更新右端点。

    示例执行过程

    [[1,3],[2,6],[8,10],[15,18]] 中，[1,3] 和 [2,6] 重叠，合并为 [1,6]。

    时间复杂度

    O(n log n)，主要来自排序。

    空间复杂度

    O(n)，答案数组最多保存所有区间。

    易错点

    判断重叠时用 start <= ans[-1][1]，端点相接也要合并，例如 [1,4] 和 [4,5]。

    ACM 模式输入输出改写

    输入第一项为区间数量 n，后面每两个整数表示一个区间。

    acm.py
    import sys

def merge_intervals(intervals):
    intervals.sort(key=lambda item: item[0])
    ans = []
    for start, end in intervals:
        if not ans or start > ans[-1][1]:
            ans.append([start, end])
        else:
            ans[-1][1] = max(ans[-1][1], end)
    return ans

data = list(map(int, sys.stdin.read().strip().split()))
n = data[0]
intervals = [[data[i], data[i + 1]] for i in range(1, 2 * n, 2)]
for start, end in merge_intervals(intervals):
    print(start, end)
