    Array / Two Pointers

    189. Rotate Array

    把数组向右轮转 k 步。

    Medium双指针数组

    题目白话解释

    数组最后 k 个元素会被挪到前面，其余元素整体后移。例如 [1,2,3,4,5,6,7] 右转 3 步变成 [5,6,7,1,2,3,4]。

    核心思路一句话

    三次反转：整体反转，再分别反转前 k 个和后 n-k 个。

    为什么会想到这个方法

    右转后的前半段来自原数组尾部。整体反转能先把尾部放到前面，但内部顺序反了；再分别反转两段即可恢复段内顺序。

    Python 前置知识

    • k %= n 可以处理 k 大于数组长度的情况。
    • 双指针交换可实现原地反转。
    • LeetCode 原题要求原地修改，不需要返回数组。

    解题步骤

    1. 计算有效轮转次数 k %= n。
    2. 反转整个数组。
    3. 反转前 k 个元素。
    4. 反转后 n-k 个元素。

    完整 Python 代码

    leetcode.py
    class Solution:
    def rotate(self, nums: list[int], k: int) -> None:
        n = len(nums)
        k %= n

        def reverse(left, right):
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1

        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)

    逐行解释

    • k %= n：轮转 n 次等于没转。
    • reverse：用左右指针反转一段区间。
    • 第一次反转把尾部元素放到前面。
    • 第二、三次反转修正两段内部顺序。

    示例执行过程

    [1,2,3,4,5,6,7] 整体反转为 [7,6,5,4,3,2,1]，前 3 个反转为 [5,6,7,4,3,2,1]，后半段反转后得到 [5,6,7,1,2,3,4]。

    时间复杂度

    O(n)，三次反转仍是线性。

    空间复杂度

    O(1)，原地修改。

    易错点

    一定要先做 k %= n，否则 k 大于数组长度时区间会错。

    ACM 模式输入输出改写

    输入第一行 n k，第二行数组，输出轮转后的数组。

    acm.py
    import sys

def rotate(nums, k):
    n = len(nums)
    k %= n
    def reverse(left, right):
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
    reverse(0, n - 1)
    reverse(0, k - 1)
    reverse(k, n - 1)
    return nums

data = sys.stdin.read().strip().split()
n, k = map(int, data[:2])
nums = list(map(int, data[2:2 + n]))
print(*rotate(nums, k))
