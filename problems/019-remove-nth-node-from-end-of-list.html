    Linked List

    19. Remove Nth Node From End of List

    删除链表倒数第 n 个节点。

    Medium链表双指针

    题目白话解释

    倒数第 n 个不好直接定位，可以让两个指针先拉开 n 步距离，再一起走。

    核心思路一句话

    快指针先走 n 步，随后快慢指针同步移动，慢指针会停在待删节点前面。

    为什么会想到这个方法

    当快指针到达链表末尾时，慢指针和快指针之间仍相差 n 步，因此慢指针正好在删除位置前。

    Python 前置知识

    • 删除头节点时需要虚拟头节点。
    • 快慢指针可以定位倒数位置。
    • 删除节点就是 prev.next = prev.next.next。

    解题步骤

    1. 创建 dummy 指向 head。
    2. 让 fast 从 dummy 先走 n 步。
    3. 让 fast 和 slow 一起走，直到 fast.next 为空。
    4. 此时 slow.next 是要删除的节点。
    5. 跳过该节点并返回 dummy.next。

    完整 Python 代码

    leetcode.py
    class Solution:
    def removeNthFromEnd(self, head, n):
        dummy = ListNode(0, head)
        slow = fast = dummy

        for _ in range(n):
            fast = fast.next

        while fast.next:
            slow = slow.next
            fast = fast.next

        slow.next = slow.next.next
        return dummy.next

    逐行解释

    • dummy 让删除头节点也统一处理。
    • 先让 fast 领先 n 步。
    • while fast.next 结束时，slow 在待删节点前面。
    • slow.next = slow.next.next 完成删除。

    示例执行过程

    [1,2,3,4,5] 删除倒数第 2 个，结果是 [1,2,3,5]。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 删除头节点必须考虑。
    • 快指针从 dummy 开始，循环条件才容易统一。
    • 不要把 slow 停在待删节点本身，否则不好删除。

    ACM 模式输入输出改写

    数组形式输入时，删除下标 len(nums) - n 的元素即可。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
nums = list(map(int, input().split()))
n = int(input())

idx = len(nums) - n
nums.pop(idx)
print(*nums)
