    Binary Tree / DFS

    101. Symmetric Tree

    判断一棵二叉树是否关于中心轴对称。

    Easy二叉树DFSBFS

    题目白话解释

    左子树和右子树要像照镜子一样：左边的左节点要对应右边的右节点，左边的右节点要对应右边的左节点。

    核心思路一句话

    写一个函数比较两个节点是否互为镜像。

    为什么会想到这个方法

    对称不是普通的“左右相同”，而是交叉相同。递归比较时必须拿左树的左边对右树的右边，左树的右边对右树的左边。

    Python 前置知识

    • 递归函数可以接收两个节点参数。
    • return a and b 表示两个条件都必须成立。
    • 空节点也要参与结构判断。

    解题步骤

    1. 空树视为对称。
    2. 比较根节点的左右子树是否镜像。
    3. 两个节点都空返回 True。
    4. 只有一个空或值不同返回 False。
    5. 交叉递归比较。

    完整 Python 代码

    leetcode.py
    class Solution:
    def isSymmetric(self, root) -> bool:
        def is_mirror(left, right):
            if left is None and right is None:
                return True
            if left is None or right is None:
                return False
            if left.val != right.val:
                return False
            return is_mirror(left.left, right.right) and is_mirror(left.right, right.left)

        return True if root is None else is_mirror(root.left, root.right)

    逐行解释

    • is_mirror：判断两个子树是否互为镜像。
    • 两个都空说明这一侧结构对称。
    • 只有一个空说明结构不对称。
    • left.val != right.val：镜像位置值必须相同。
    • 交叉比较左右孩子。

    示例执行过程

    [1,2,2,3,4,4,3] 中，左子树的 3 对应右子树的 3，左子树的 4 对应右子树的 4，因此对称。

    时间复杂度

    O(n)，每个节点最多比较一次。

    空间复杂度

    O(h)，递归栈高度取决于树高。

    易错点

    不要写成左对左、右对右，那是在判断两棵树相同，不是在判断镜像对称。

    ACM 模式输入输出改写

    输入一行层序数组，空节点用 null 表示。

    acm.py
    import sys
from collections import deque

class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def build_tree(values):
    if not values or values[0] == "null":
        return None
    root = TreeNode(values[0])
    queue = deque([root])
    i = 1
    while queue and i < len(values):
        node = queue.popleft()
        if i < len(values) and values[i] != "null":
            node.left = TreeNode(values[i])
            queue.append(node.left)
        i += 1
        if i < len(values) and values[i] != "null":
            node.right = TreeNode(values[i])
            queue.append(node.right)
        i += 1
    return root

def is_mirror(left, right):
    if left is None and right is None:
        return True
    if left is None or right is None:
        return False
    return left.val == right.val and is_mirror(left.left, right.right) and is_mirror(left.right, right.left)

values = sys.stdin.read().strip().split()
root = build_tree(values)
ok = True if root is None else is_mirror(root.left, root.right)
print("true" if ok else "false")
