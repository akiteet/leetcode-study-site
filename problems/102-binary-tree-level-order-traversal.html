    Binary Tree / BFS

    102. Binary Tree Level Order Traversal

    按层从左到右遍历二叉树。

    Medium 二叉树 队列 BFS

    核心思路

    用队列保存当前要访问的节点。每次先记录当前队列长度，这个长度就是本层节点数。

    题目理解

    • 返回结果是二维列表，每一层一个列表。
    • 同一层按从左到右顺序输出。
    • 空树返回空列表。

    解题过程

    1. 根节点为空则返回 []。
    2. 把根节点加入队列。
    3. 循环处理队列，每轮处理一整层。
    4. 弹出节点，记录值，把左右孩子加入队列。
    5. 把本层列表加入答案。

    Python 代码块

    leetcode.py
    from collections import deque

class Solution:
    def levelOrder(self, root) -> list[list[int]]:
        if root is None:
            return []

        ans = []
        queue = deque([root])

        while queue:
            level = []
            for _ in range(len(queue)):
                node = queue.popleft()
                level.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            ans.append(level)

        return ans

    逐行解释

    • deque([root])：队列适合从左侧弹出节点。
    • for _ in range(len(queue))：固定当前层节点数。
    • popleft()：按先进先出顺序访问。
    • level.append(node.val)：收集当前层的节点值。
    • ans.append(level)：一层处理完再加入答案。

    复杂度分析

    时间复杂度 O(n)，每个节点入队出队一次。空间复杂度 O(w)，w 是树的最大宽度。

    易错点

    处理每层前要先固定队列长度，否则新加入的下一层节点会被混进当前层。

    ACM 模式改写

    输入层序数组，空节点用 null 表示。每行输出一层。

    acm.py
    import sys
from collections import deque

class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def build_tree(values):
    if not values or values[0] == "null":
        return None
    root = TreeNode(values[0])
    queue = deque([root])
    i = 1
    while queue and i < len(values):
        node = queue.popleft()
        if i < len(values) and values[i] != "null":
            node.left = TreeNode(values[i])
            queue.append(node.left)
        i += 1
        if i < len(values) and values[i] != "null":
            node.right = TreeNode(values[i])
            queue.append(node.right)
        i += 1
    return root

def level_order(root):
    if root is None:
        return []
    ans = []
    queue = deque([root])
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(str(node.val))
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        ans.append(level)
    return ans

values = sys.stdin.read().strip().split()
root = build_tree(values)
for level in level_order(root):
    print(" ".join(level))
