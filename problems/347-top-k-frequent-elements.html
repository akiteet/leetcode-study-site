    Hash Table / Heap

    347. Top K Frequent Elements

    返回数组中出现频率最高的 k 个元素。

    Medium哈希表堆计数

    题目白话解释

    先数每个数字出现了几次，再挑出次数最多的 k 个数字。

    核心思路一句话

    用字典计数，再用小顶堆保留频率最高的 k 个元素。

    为什么会想到这个方法

    题目关心“频率”，所以第一步一定是计数。只要前 k 个，不需要完整排序全部元素，用大小为 k 的堆能减少不必要排序。

    Python 前置知识

    • dict 或 Counter 用于计数。
    • heapq 默认是小顶堆。
    • heappush 入堆，heappop 弹出最小元素。

    解题步骤

    1. 遍历数组统计频率。
    2. 创建空堆。
    3. 把 (频率, 数字) 放入堆。
    4. 堆长度超过 k 时弹出频率最低的元素。
    5. 最后堆中剩下的数字就是答案。

    完整 Python 代码

    leetcode.py
    import heapq
from collections import Counter

class Solution:
    def topKFrequent(self, nums: list[int], k: int) -> list[int]:
        count = Counter(nums)
        heap = []

        for num, freq in count.items():
            heapq.heappush(heap, (freq, num))
            if len(heap) > k:
                heapq.heappop(heap)

        return [num for freq, num in heap]

    逐行解释

    • Counter(nums)：得到数字到出现次数的映射。
    • heap = []：用列表作为堆。
    • (freq, num)：让堆按频率比较大小。
    • len(heap) > k：只保留 k 个候选。
    • heappop：弹掉当前频率最低的元素。

    示例执行过程

    输入 [1,1,1,2,2,3], k = 2。计数得到 1 出现 3 次、2 出现 2 次、3 出现 1 次。堆最终留下 1 和 2。

    时间复杂度

    O(n log k)，计数是 O(n)，每个不同元素最多进出大小为 k 的堆。

    空间复杂度

    O(n + k)，计数字典最多保存 n 个不同元素，堆保存 k 个元素。

    易错点

    返回顺序通常不要求固定；如果平台要求特定顺序，需要再排序结果。

    ACM 模式输入输出改写

    输入第一行 n k，第二行 n 个整数，输出前 k 高频元素。

    acm.py
    import sys
import heapq
from collections import Counter

def top_k_frequent(nums, k):
    count = Counter(nums)
    heap = []
    for num, freq in count.items():
        heapq.heappush(heap, (freq, num))
        if len(heap) > k:
            heapq.heappop(heap)
    return [num for freq, num in heap]

data = sys.stdin.read().strip().split()
n, k = map(int, data[:2])
nums = list(map(int, data[2:2 + n]))
print(*top_k_frequent(nums, k))
