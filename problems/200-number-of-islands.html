    200. Number of Islands

    统计网格中由上下左右相邻陆地组成的岛屿数量。

    Medium DFS BFS 矩阵

    核心思路

    遍历网格，遇到一块未访问的陆地就说明发现新岛屿。用 DFS 把这座岛的所有陆地都标记为已访问。

    题目理解

    • "1" 表示陆地，"0" 表示水。
    • 只看上下左右相邻，不看对角线。
    • 每发现一个新的未访问陆地，岛屿数量加 1。

    解题过程

    1. 遍历每个格子。
    2. 如果当前格子是 "1"，岛屿数量加 1。
    3. 从该格子出发做 DFS。
    4. DFS 中把访问过的陆地改成 "0"。
    5. 继续遍历，直到所有格子处理完。

    Python 代码块

    leetcode.py
    class Solution:
    def numIslands(self, grid: list[list[str]]) -> int:
        if not grid:
            return 0

        rows, cols = len(grid), len(grid[0])

        def dfs(r, c):
            if r < 0 or r >= rows or c < 0 or c >= cols:
                return
            if grid[r][c] != "1":
                return

            grid[r][c] = "0"
            dfs(r + 1, c)
            dfs(r - 1, c)
            dfs(r, c + 1)
            dfs(r, c - 1)

        count = 0
        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == "1":
                    count += 1
                    dfs(r, c)

        return count

    逐行解释

    • rows, cols：记录矩阵边界。
    • dfs(r, c)：从一个陆地格子扩散访问整座岛。
    • grid[r][c] = "0"：标记已访问，避免重复统计。
    • 四次递归：分别访问上下左右。
    • count += 1：每次从未访问陆地启动 DFS，代表一座新岛。

    复杂度分析

    时间复杂度 O(m * n)，每个格子最多访问一次。空间复杂度 O(m * n)，最坏情况下递归栈覆盖整张网格。

    易错点

    必须及时标记已访问，否则 DFS 会在相邻陆地之间来回递归，导致重复访问甚至递归过深。

    ACM 模式改写

    输入第一行 m n，后面 m 行每行一个 01 字符串，输出岛屿数量。

    acm.py
    import sys

def num_islands(grid):
    if not grid:
        return 0
    rows, cols = len(grid), len(grid[0])

    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols:
            return
        if grid[r][c] != "1":
            return
        grid[r][c] = "0"
        dfs(r + 1, c)
        dfs(r - 1, c)
        dfs(r, c + 1)
        dfs(r, c - 1)

    count = 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == "1":
                count += 1
                dfs(r, c)
    return count

lines = sys.stdin.read().strip().splitlines()
m, n = map(int, lines[0].split())
grid = [list(lines[i + 1].strip()) for i in range(m)]
print(num_islands(grid))
