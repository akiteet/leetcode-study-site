    Sliding Window / Hash Table

    438. Find All Anagrams in a String

    在字符串 s 中找出所有 p 的字母异位词起始下标。

    Medium滑动窗口哈希表字符串

    题目白话解释

    拿一个长度和 p 一样的窗口在 s 上滑动，只要窗口里的字符次数和 p 完全一样，就记录窗口起点。

    核心思路一句话

    固定窗口长度为 len(p)，比较窗口计数和目标计数。

    为什么会想到这个方法

    异位词只关心字符种类和次数，不关心顺序。而子串长度必须和 p 相同，所以固定长度滑动窗口最直接。

    Python 前置知识

    • Counter 可以统计字符次数。
    • 窗口右进左出时要同步更新计数。
    • 计数为 0 的键可以删除，方便比较字典。

    解题步骤

    1. 统计 p 的字符次数。
    2. 遍历 s，把右侧字符加入窗口。
    3. 如果窗口长度超过 len(p)，移出左侧字符。
    4. 当窗口长度等于 len(p) 且计数相同，记录起点。

    完整 Python 代码

    leetcode.py
    from collections import Counter

class Solution:
    def findAnagrams(self, s: str, p: str) -> list[int]:
        need = Counter(p)
        window = Counter()
        ans = []
        left = 0

        for right, ch in enumerate(s):
            window[ch] += 1

            if right - left + 1 > len(p):
                out = s[left]
                window[out] -= 1
                if window[out] == 0:
                    del window[out]
                left += 1

            if right - left + 1 == len(p) and window == need:
                ans.append(left)

        return ans

    逐行解释

    • need：目标字符串的字符计数。
    • window：当前窗口的字符计数。
    • window[ch] += 1：右侧字符进入窗口。
    • del window[out]：删除 0 次字符，保证字典比较准确。
    • ans.append(left)：记录异位词起点。

    示例执行过程

    s = "cbaebabacd", p = "abc"。窗口 "cba" 和 "bac" 都与 abc 计数一致，所以返回 [0,6]。

    时间复杂度

    O(n)，如果字符集固定为小写英文字母，比较计数可看作常数。

    空间复杂度

    O(1)，小写字母最多 26 种；泛化字符集下为 O(k)。

    易错点

    窗口长度必须固定为 len(p)。长度不同即使包含相同字符，也不是本题要找的异位词子串。

    ACM 模式输入输出改写

    输入两行字符串 s 和 p，输出所有起始下标。

    acm.py
    import sys
from collections import Counter

def find_anagrams(s, p):
    need = Counter(p)
    window = Counter()
    ans = []
    left = 0
    for right, ch in enumerate(s):
        window[ch] += 1
        if right - left + 1 > len(p):
            out = s[left]
            window[out] -= 1
            if window[out] == 0:
                del window[out]
            left += 1
        if right - left + 1 == len(p) and window == need:
            ans.append(left)
    return ans

lines = sys.stdin.read().strip().split()
s, p = lines[0], lines[1]
print(*find_anagrams(s, p))
