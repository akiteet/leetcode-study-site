    45. Jump Game II

    求从起点跳到终点所需的最少跳跃次数。

    Medium贪心数组

    题目白话解释

    这次一定可以到终点，目标是用尽量少的跳数到达。

    核心思路一句话

    按层扩展当前跳数能覆盖的范围，到达当前边界时必须再跳一次。

    为什么会想到这个方法

    一次跳跃能覆盖一个区间，下一跳应该让下一层覆盖尽量远；这和 BFS 分层类似，但只需要维护边界。

    Python 前置知识

    • end 表示当前跳数能到达的最右边界。
    • farthest 表示下一跳能扩展到的最远位置。
    • 遍历到 len(nums)-2 即可，避免在终点多跳。

    解题步骤

    1. 初始化 steps = 0, end = 0, farthest = 0。
    2. 遍历到倒数第二个位置。
    3. 不断更新 farthest。
    4. 当 i == end，说明当前层走完，跳数加一。
    5. 把 end 更新为 farthest。

    完整 Python 代码

    leetcode.py
    class Solution:
    def jump(self, nums: list[int]) -> int:
        steps = 0
        end = 0
        farthest = 0

        for i in range(len(nums) - 1):
            farthest = max(farthest, i + nums[i])
            if i == end:
                steps += 1
                end = farthest

        return steps

    逐行解释

    • farthest 收集当前跳数范围内能扩展到的最远点。
    • i == end 表示这一跳覆盖的范围已经扫描完。
    • 此时必须增加一次跳跃。
    • 终点不需要再往外跳，所以遍历到倒数第二个位置。

    示例执行过程

    [2,3,1,1,4] 最少跳两次：0 跳到 1，1 跳到 4。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 循环不要遍历到最后一个位置，否则可能多加一步。
    • 本题求最少次数，不是判断能否到达。
    • end 和 farthest 含义不要混淆。

    ACM 模式输入输出改写

    输入数组，输出最少跳跃次数。

    acm.py
    # input() + split() + map() 读取数组，print() 输出答案
nums = list(map(int, input().split()))

steps = 0
end = 0
farthest = 0
for i in range(len(nums) - 1):
    farthest = max(farthest, i + nums[i])
    if i == end:
        steps += 1
        end = farthest

print(steps)
