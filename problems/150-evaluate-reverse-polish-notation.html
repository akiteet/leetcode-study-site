    150. Evaluate Reverse Polish Notation

    计算逆波兰表达式的值。

    Medium栈数组

    题目白话解释

    表达式里数字在前，运算符在后。例如 ["2","1","+","3","*"] 表示 (2 + 1) * 3。

    核心思路一句话

    遇到数字入栈，遇到运算符就弹出两个数字计算后再入栈。

    为什么会想到这个方法

    逆波兰表达式的特点是运算符出现时，它需要的两个操作数一定已经在前面出现。后出现的数字先参与运算，正好符合栈的后进先出。

    Python 前置知识

    • str.lstrip("-").isdigit() 可判断整数 token。
    • 整数除法要向 0 截断，Python 的 int(a / b) 满足这题。
    • 弹栈顺序要注意：先弹出的是右操作数。

    解题步骤

    1. 创建空栈。
    2. 遍历 tokens。
    3. 数字转成 int 入栈。
    4. 运算符弹出右操作数和左操作数。
    5. 计算结果入栈，最后栈顶就是答案。

    完整 Python 代码

    leetcode.py
    class Solution:
    def evalRPN(self, tokens: list[str]) -> int:
        stack = []

        for token in tokens:
            if token not in {"+", "-", "*", "/"}:
                stack.append(int(token))
                continue

            b = stack.pop()
            a = stack.pop()
            if token == "+":
                stack.append(a + b)
            elif token == "-":
                stack.append(a - b)
            elif token == "*":
                stack.append(a * b)
            else:
                stack.append(int(a / b))

        return stack[-1]

    逐行解释

    • stack：保存还没参与完计算的数字。
    • token not in ...：不是运算符就是数字。
    • b = stack.pop()：右操作数先弹出。
    • a = stack.pop()：左操作数后弹出。
    • int(a / b)：除法结果向 0 截断。

    示例执行过程

    ["2","1","+","3","*"]：2、1 入栈，遇到 + 得到 3；3 入栈，遇到 * 得到 9。

    时间复杂度

    O(n)，每个 token 处理一次。

    空间复杂度

    O(n)，最坏情况下很多数字先入栈。

    易错点

    减法和除法不能交换操作数，必须是后弹出的 a 在前、先弹出的 b 在后。

    ACM 模式输入输出改写

    输入第一项为 token 数量 n，后面跟着表达式 token，输出计算结果。

    acm.py
    import sys

def eval_rpn(tokens):
    stack = []
    for token in tokens:
        if token not in {"+", "-", "*", "/"}:
            stack.append(int(token))
        else:
            b = stack.pop()
            a = stack.pop()
            if token == "+":
                stack.append(a + b)
            elif token == "-":
                stack.append(a - b)
            elif token == "*":
                stack.append(a * b)
            else:
                stack.append(int(a / b))
    return stack[-1]

data = sys.stdin.read().strip().split()
n = int(data[0])
tokens = data[1:1 + n]
print(eval_rpn(tokens))
