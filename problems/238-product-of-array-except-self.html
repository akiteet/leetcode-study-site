    238. Product of Array Except Self

    返回每个位置除自身以外所有数字的乘积。

    Medium数组前后缀

    题目白话解释

    对每个位置，计算左边所有数乘积乘以右边所有数乘积，不能使用除法。

    核心思路一句话

    先把左侧乘积写入答案，再从右往左乘上右侧乘积。

    为什么会想到这个方法

    当前位置不包含自己，所以答案天然拆成左边乘积和右边乘积。两次遍历刚好能分别得到这两个部分。

    Python 前置知识

    • [1] * n 创建初始答案数组。
    • 变量 prefix 可滚动保存左侧乘积。
    • 变量 suffix 可滚动保存右侧乘积。

    解题步骤

    1. 创建答案数组，初始都为 1。
    2. 从左到右，把当前位置左侧乘积写入答案。
    3. 从右到左，用右侧乘积乘到答案上。
    4. 返回答案数组。

    完整 Python 代码

    leetcode.py
    class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        n = len(nums)
        ans = [1] * n

        prefix = 1
        for i in range(n):
            ans[i] = prefix
            prefix *= nums[i]

        suffix = 1
        for i in range(n - 1, -1, -1):
            ans[i] *= suffix
            suffix *= nums[i]

        return ans

    逐行解释

    • ans[i] = prefix：先放入当前位置左边所有数的乘积。
    • prefix *= nums[i]：当前位置加入后续位置的左侧乘积。
    • suffix：从右侧滚动维护。
    • ans[i] *= suffix：把右边乘积补上。

    示例执行过程

    [1,2,3,4] 左侧乘积先得到 [1,1,2,6]，再乘右侧乘积后得到 [24,12,8,6]。

    时间复杂度

    O(n)，两次遍历。

    空间复杂度

    除输出数组外为 O(1)。

    易错点

    不要用除法，这题常见限制就是要求不使用除法，并且数组里可能有 0。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着数组，输出结果数组。

    acm.py
    import sys

def product_except_self(nums):
    n = len(nums)
    ans = [1] * n
    prefix = 1
    for i in range(n):
        ans[i] = prefix
        prefix *= nums[i]
    suffix = 1
    for i in range(n - 1, -1, -1):
        ans[i] *= suffix
        suffix *= nums[i]
    return ans

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
print(*product_except_self(nums))
