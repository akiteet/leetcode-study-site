    Two Pointers / String

    125. Valid Palindrome

    只保留字母数字并忽略大小写后，判断字符串是否为回文。

    Easy 双指针 字符串

    核心思路

    左右指针分别从两端向中间移动。遇到非字母数字就跳过，遇到有效字符就转小写后比较。

    题目理解

    • 标点、空格等非字母数字字符需要忽略。
    • 大小写不敏感，A 和 a 视为相同。
    • 空字符串或过滤后为空，也视为回文。

    解题过程

    1. 设置左右指针 left 和 right。
    2. 左指针跳过非字母数字字符。
    3. 右指针跳过非字母数字字符。
    4. 比较两个有效字符的小写形式。
    5. 如果不相等返回 False，否则继续向中间移动。

    Python 代码块

    leetcode.py
    class Solution:
    def isPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1

        while left < right:
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1

            if s[left].lower() != s[right].lower():
                return False

            left += 1
            right -= 1

        return True

    逐行解释

    • left, right：分别指向字符串两端。
    • isalnum()：判断字符是否为字母或数字。
    • lower()：统一大小写后再比较。
    • left += 1、right -= 1：当前字符匹配后继续向中间收缩。
    • return True：全部比较通过。

    复杂度分析

    时间复杂度 O(n)，每个字符最多被访问常数次。空间复杂度 O(1)。

    易错点

    跳过无效字符时也要判断 left < right，避免指针越界或重复比较。

    ACM 模式改写

    输入一整行字符串，输出 true 或 false。

    acm.py
    import sys

def is_palindrome(s):
    left, right = 0, len(s) - 1
    while left < right:
        while left < right and not s[left].isalnum():
            left += 1
        while left < right and not s[right].isalnum():
            right -= 1
        if s[left].lower() != s[right].lower():
            return False
        left += 1
        right -= 1
    return True

s = sys.stdin.readline().rstrip("\n")
print("true" if is_palindrome(s) else "false")
