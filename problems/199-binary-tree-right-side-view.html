    Binary Tree

    199. Binary Tree Right Side View

    从右侧看二叉树，返回每一层最右边的节点。

    Medium二叉树BFSDFS

    题目白话解释

    每层只保留最右侧那个能被看到的节点。

    核心思路一句话

    层序遍历，每一层最后弹出的节点就是右视图节点。

    为什么会想到这个方法

    右视图按层输出，BFS 天然按层处理；只要知道每层的最后一个节点即可。

    Python 前置知识

    • deque 适合做队列。
    • 层序遍历要先记录当前层大小。
    • 每层遍历结束时的节点就是该层最右节点。

    解题步骤

    1. 如果根为空，返回空列表。
    2. 把根节点放入队列。
    3. 每轮先取当前队列长度作为层大小。
    4. 弹出这一层所有节点并加入下一层。
    5. 当本层索引是最后一个时，把值加入答案。

    完整 Python 代码

    leetcode.py
    from collections import deque

class Solution:
    def rightSideView(self, root):
        if not root:
            return []

        ans = []
        queue = deque([root])
        while queue:
            size = len(queue)
            for i in range(size):
                node = queue.popleft()
                if i == size - 1:
                    ans.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
        return ans

    逐行解释

    • size 固定当前层节点数。
    • i == size - 1 说明当前节点是这一层最后一个。
    • 先加左再加右也可以，因为最后访问的是右侧节点。
    • 队列保存下一层节点。

    示例执行过程

    [1,2,3,null,5,null,4] 的右视图是 [1,3,4]。

    时间复杂度

    O(n)。

    空间复杂度

    O(w)，队列最多保存一层节点。

    易错点

    • 不要把整棵树最后一个节点当答案。
    • 每层都要重新计算 size。
    • 空树返回空列表。

    ACM 模式输入输出改写

    层序数组输入时，可以按数组下标模拟层级并输出每层最右非空节点。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
vals = input().split()
# 常用整数行：nums = list(map(int, input().split()))

ans = []
level_start = 0
level_size = 1
while level_start < len(vals):
    level = vals[level_start:level_start + level_size]
    for x in reversed(level):
        if x != "null":
            ans.append(int(x))
            break
    level_start += level_size
    level_size *= 2

print(*ans)
