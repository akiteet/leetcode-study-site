    Dynamic Programming

    118. Pascal's Triangle

    生成杨辉三角的前 numRows 行。

    Easy动态规划入门数组

    题目白话解释

    每一行两边都是 1，中间的数等于上一行相邻两个数之和。

    核心思路一句话

    从上到下构造，每一行依赖上一行。

    为什么会想到这个方法

    当前行的每个中间位置都有明确来源：上一行左上和右上。这个局部递推就是动态规划的入门形式。

    Python 前置知识

    • 二维列表可以保存所有行。
    • 第 i 行长度是 i + 1。
    • 中间位置范围是 1 到 i - 1。

    解题步骤

    1. 创建空答案列表。
    2. 逐行创建长度为 i + 1、全是 1 的列表。
    3. 中间位置用上一行两个数相加填充。
    4. 把当前行加入答案。
    5. 返回答案。

    完整 Python 代码

    leetcode.py
    class Solution:
    def generate(self, numRows: int) -> list[list[int]]:
        ans = []
        for i in range(numRows):
            row = [1] * (i + 1)
            for j in range(1, i):
                row[j] = ans[i - 1][j - 1] + ans[i - 1][j]
            ans.append(row)
        return ans

    逐行解释

    • row = [1] * (i + 1) 先放好两边的 1。
    • j 只遍历中间位置。
    • ans[i - 1] 是上一行。
    • 每行算完加入 ans。

    示例执行过程

    numRows = 5 时输出 [1]、[1,1]、[1,2,1]、[1,3,3,1]、[1,4,6,4,1]。

    时间复杂度

    O(numRows^2)。

    空间复杂度

    O(numRows^2)，需要返回所有行。

    易错点

    • 不要修改上一行。
    • 第一行和第二行没有中间位置。
    • 题目要返回所有行，不是只返回最后一行。

    # input() 读取；split() / map() 可处理多数字输入；print() 输出

    for j in range(1, i):
        row[j] = ans[i - 1][j - 1] + ans[i - 1][j]
