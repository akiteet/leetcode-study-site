    230. Kth Smallest Element in a BST

    在二叉搜索树中找到第 k 小的值。

    Medium二叉树二叉搜索树中序遍历

    题目白话解释

    BST 的中序遍历是升序，所以第 k 个访问到的节点就是第 k 小。

    核心思路一句话

    中序遍历 BST，数到第 k 个时返回。

    为什么会想到这个方法

    二叉搜索树的左根右顺序天然从小到大，不需要额外排序。

    Python 前置知识

    • BST 中序遍历结果有序。
    • 迭代写法可以用栈模拟递归。
    • 计数从 1 到 k。

    解题步骤

    1. 用栈一路把左孩子入栈。
    2. 弹出栈顶表示访问当前最小未访问节点。
    3. k -= 1。
    4. 当 k == 0 时返回当前值。
    5. 转向当前节点右子树。

    完整 Python 代码

    leetcode.py
    class Solution:
    def kthSmallest(self, root, k):
        stack = []
        cur = root

        while cur or stack:
            while cur:
                stack.append(cur)
                cur = cur.left

            cur = stack.pop()
            k -= 1
            if k == 0:
                return cur.val
            cur = cur.right

    逐行解释

    • 内层循环不断走到最左边。
    • stack.pop() 得到当前最小的未访问节点。
    • 每访问一个节点就让 k 减一。
    • 访问根后转向右子树。

    示例执行过程

    BST [3,1,4,null,2] 的中序是 [1,2,3,4]，第 1 小是 1。

    时间复杂度

    平均 O(h+k)，最坏 O(n)。

    空间复杂度

    O(h)。

    易错点

    • 不要对所有节点排序，BST 已经提供顺序。
    • 第 k 小不是下标 k，而是第 k 个，计数要减到 0。
    • 中序顺序必须是左根右。

    ACM 模式输入输出改写

    ACM 可先输入 BST 的层序数组，再输入 k，输出第 k 小。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
vals = input().split()
k = int(input())
# 常用整数行：nums = list(map(int, input().split()))

def inorder(i):
    if i >= len(vals) or vals[i] == "null":
        return []
    return inorder(2 * i + 1) + [int(vals[i])] + inorder(2 * i + 2)

arr = inorder(0)
print(arr[k - 1])
