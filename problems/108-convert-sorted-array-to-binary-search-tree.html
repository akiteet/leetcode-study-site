    Binary Tree

    108. Convert Sorted Array to Binary Search Tree

    把升序数组转换成高度平衡的二叉搜索树。

    Easy二叉树二叉搜索树递归

    题目白话解释

    数组已经有序，要让树尽量平衡，就应该每次选中间元素当根。

    核心思路一句话

    递归选择中点作为根，左半部分建左子树，右半部分建右子树。

    为什么会想到这个方法

    二叉搜索树要求左小右大；有序数组天然满足。选中点可以让左右节点数量接近，从而高度平衡。

    Python 前置知识

    • 二叉搜索树左子树小于根，右子树大于根。
    • 递归区间常用 left, right。
    • 中点写作 (left + right) // 2。

    解题步骤

    1. 定义 build(left, right)。
    2. 如果 left > right，返回空。
    3. 取中点 mid。
    4. 创建根节点 nums[mid]。
    5. 递归构造左右子树。

    完整 Python 代码

    leetcode.py
    class Solution:
    def sortedArrayToBST(self, nums):
        def build(left, right):
            if left > right:
                return None
            mid = (left + right) // 2
            root = TreeNode(nums[mid])
            root.left = build(left, mid - 1)
            root.right = build(mid + 1, right)
            return root

        return build(0, len(nums) - 1)

    逐行解释

    • build 负责把一个数组区间建成树。
    • 中点作为当前根节点。
    • 左区间生成左子树。
    • 右区间生成右子树。

    示例执行过程

    [-10,-3,0,5,9] 可选 0 为根，左右两边分别递归建树。

    时间复杂度

    O(n)。

    空间复杂度

    O(log n)，平衡树递归深度。

    易错点

    • 不要总选第一个元素，否则会退化成链表。
    • 递归边界是 left > right。
    • 返回的是树根，不是数组。

    ACM 模式输入输出改写

    ACM 可输出构造后树的前序遍历，方便检查结构。

    acm.py
    # input() 读取一行，split() 切分，map() 转整数，print() 输出结果
nums = list(map(int, input().split()))

def preorder(left, right):
    if left > right:
        return []
    mid = (left + right) // 2
    return [nums[mid]] + preorder(left, mid - 1) + preorder(mid + 1, right)

print(*preorder(0, len(nums) - 1))
