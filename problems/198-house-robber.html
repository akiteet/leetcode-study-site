    Dynamic Programming

    198. House Robber

    不能偷相邻房子，求能偷到的最大金额。

    Medium动态规划入门数组

    题目白话解释

    每间房子有钱，但相邻两间不能同时偷。你需要在不触发警报的前提下拿到最多钱。

    核心思路一句话

    到当前房子时，选择“偷当前 + 前前最优”或“不偷当前 + 前一个最优”。

    为什么会想到这个方法

    每个位置的选择会影响相邻位置，当前最优依赖前面状态，这是动态规划的特征。

    Python 前置知识

    • max(a, b) 表示在两种选择里取更优。
    • 滚动变量可保存前一个和前两个状态。
    • 遍历数组时每个值代表当前房子金额。

    解题步骤

    1. prev2 表示前前个房子的最优。
    2. prev1 表示前一个房子的最优。
    3. 遍历当前金额。
    4. 当前最优为 max(prev1, prev2 + money)。
    5. 滚动更新状态。

    完整 Python 代码

    leetcode.py
    class Solution:
    def rob(self, nums: list[int]) -> int:
        prev2 = 0
        prev1 = 0

        for money in nums:
            current = max(prev1, prev2 + money)
            prev2, prev1 = prev1, current

        return prev1

    逐行解释

    • prev2：不相邻的上一个可组合状态。
    • prev1：前一个位置的最大收益。
    • prev2 + money：偷当前房子。
    • prev1：不偷当前房子。
    • prev1 最终保存全局最优。

    示例执行过程

    [2,7,9,3,1]：偷 2 和 9 再加 1，总额 12，是不偷相邻房子的最大值。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    不是简单选择所有奇数位或偶数位，因为局部金额大小会改变最优组合。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着 n 个房子金额，输出最大金额。

    acm.py
    import sys

def rob(nums):
    prev2 = 0
    prev1 = 0
    for money in nums:
        current = max(prev1, prev2 + money)
        prev2, prev1 = prev1, current
    return prev1

data = sys.stdin.read().strip().split()
n = int(data[0])
nums = list(map(int, data[1:1 + n]))
print(rob(nums))
