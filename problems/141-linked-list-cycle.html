    141. Linked List Cycle

    判断链表中是否存在环。

    Easy链表快慢指针

    题目白话解释

    如果某个节点的 next 指回前面的节点，链表就会一直走不完，这就是有环。

    核心思路一句话

    快慢指针同时走，快指针如果追上慢指针就有环。

    为什么会想到这个方法

    在环里，快指针每次比慢指针多走一步，距离会不断缩短，最终一定相遇；无环时快指针会先走到空。

    Python 前置知识

    • 快指针每次走两步，慢指针每次走一步。
    • 循环条件要保证 fast 和 fast.next 都存在。
    • 快慢指针是链表判环的标准模板。

    解题步骤

    1. 初始化 slow = fast = head。
    2. 循环中慢指针走一步，快指针走两步。
    3. 如果二者相遇，返回 True。
    4. 如果快指针走到空，返回 False。

    完整 Python 代码

    leetcode.py
    class Solution:
    def hasCycle(self, head):
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False

    逐行解释

    • fast and fast.next 防止访问空节点的 next。
    • slow == fast 表示两个指针到了同一个节点。
    • 有环时不会走到空节点。
    • 无环时快指针会先到链表尾部。

    示例执行过程

    3 -&gt; 2 -&gt; 0 -&gt; -4 且 -4 指回 2，快慢指针会在环中相遇。

    时间复杂度

    O(n)。

    空间复杂度

    O(1)。

    易错点

    • 比较的是节点对象，不是节点值。
    • 循环条件不能只写 while fast。
    • 空链表和单节点无环都要返回 False。

    ACM 模式输入输出改写

    ACM 中常给一个 pos 表示尾节点连回的位置，-1 表示无环。

pos = int(input())
