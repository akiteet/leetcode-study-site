    215. Kth Largest Element in an Array

    不需要完全排序，找数组中第 k 大的元素。

    Medium堆Top K

    题目白话解释

    第 k 大不是第 k 个不同元素，而是排序后从大到小数第 k 个位置的值。

    核心思路一句话

    维护一个大小为 k 的小根堆，堆顶就是当前第 k 大。

    为什么会想到这个方法

    只关心前 k 大，没必要保存所有元素的完整顺序。小根堆可以把前 k 大里最小的放在堆顶，方便被更大的数替换。

    Python 前置知识

    • Python 的 heapq 默认是小根堆。
    • heappush 入堆，heappop 弹出最小值。
    • 堆大小保持为 k，最后堆顶就是答案。

    解题步骤

    1. 创建空堆。
    2. 遍历数组，把每个数放进堆。
    3. 如果堆大小超过 k，就弹出最小值。
    4. 遍历结束后，堆中保留最大的 k 个数。
    5. 返回堆顶。

    完整 Python 代码

    leetcode.py
    import heapq

class Solution:
    def findKthLargest(self, nums: list[int], k: int) -> int:
        heap = []
        for num in nums:
            heapq.heappush(heap, num)
            if len(heap) > k:
                heapq.heappop(heap)
        return heap[0]

    逐行解释

    • 堆中最多保留 k 个元素。
    • 超过 k 时弹出最小值，说明它不可能是前 k 大。
    • 最后剩下的是最大的 k 个数。
    • 这 k 个数里最小的就是第 k 大。

    示例执行过程

    [3,2,1,5,6,4]，k=2，第二大是 5。

    时间复杂度

    O(n log k)。

    空间复杂度

    O(k)。

    易错点

    • 第 k 大不是下标 k。
    • 重复元素也要计入排名。
    • 小根堆保留 k 个最大值，不是大根堆保留所有值。

    ACM 模式输入输出改写

    输入 n 和 k，再输入数组，输出第 k 大。

    acm.py
    # input() + split() + map() 读取，print() 输出
import heapq

n, k = map(int, input().split())
nums = list(map(int, input().split()))

heap = []
for num in nums:
    heapq.heappush(heap, num)
    if len(heap) > k:
        heapq.heappop(heap)

print(heap[0])
