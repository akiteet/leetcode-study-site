    Two Pointers

    11. Container With Most Water

    从多条竖线中选两条，使它们和 x 轴围成的容器能装最多水。

    Medium双指针数组

    题目白话解释

    每个高度是一块挡板，选左右两块挡板，水量等于较短挡板高度乘以两挡板距离。

    核心思路一句话

    左右指针从两端开始，每次移动较短的那一边。

    为什么会想到这个方法

    面积由宽度和短板共同决定。两端开始时宽度最大，之后宽度只会变小，所以只有移动短板才可能找到更高的短板来弥补宽度损失。

    Python 前置知识

    • 双指针常用 left、right 表示两端。
    • min(a, b) 可取短板高度。
    • while left < right 用于两端向中间收缩。

    解题步骤

    1. 设置左右指针。
    2. 计算当前面积。
    3. 更新最大面积。
    4. 比较左右高度，移动较短的一边。
    5. 直到两个指针相遇。

    完整 Python 代码

    leetcode.py
    class Solution:
    def maxArea(self, height: list[int]) -> int:
        left, right = 0, len(height) - 1
        best = 0

        while left < right:
            width = right - left
            water = min(height[left], height[right]) * width
            best = max(best, water)

            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return best

    逐行解释

    • left, right：从最宽的容器开始。
    • width：两条线之间的距离。
    • min(...)：水位由短板决定。
    • best：保存最大面积。
    • 移动短板：尝试找到更高的边界。

    示例执行过程

    输入 [1,8,6,2,5,4,8,3,7]。初始面积为 8，移动左侧短板后遇到高度 8，和右侧 7 形成面积 49。

    时间复杂度

    O(n)，左右指针总共移动 n 次以内。

    空间复杂度

    O(1)，只使用常数变量。

    易错点

    不要移动较高的一边。较高边不是当前瓶颈，移动它只会让宽度变小且短板不变。

    ACM 模式输入输出改写

    输入第一项为 n，后面跟着 n 个高度，输出最大面积。

    acm.py
    import sys

def max_area(height):
    left, right = 0, len(height) - 1
    best = 0
    while left < right:
        best = max(best, min(height[left], height[right]) * (right - left))
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    return best

data = sys.stdin.read().strip().split()
n = int(data[0])
height = list(map(int, data[1:1 + n]))
print(max_area(height))
